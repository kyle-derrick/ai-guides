# 第三十章 税务筹划 — 练习方法

## 为什么要通过练习学税务筹划

税务筹划是一门**实践性极强**的技能。读完理论知识后，如果不动手计算、不分析真实案例，知识永远停留在"知道"层面，无法转化为"会用"的能力。

练习的核心目标有三个：

1. **建立税感**：看到一笔收入或交易，本能地想到涉及哪些税种、税率多少、有没有优惠政策
2. **形成肌肉记忆**：个税计算、扣除项目、优惠条件等高频知识点做到信手拈来
3. **培养筹划思维**：从"事后算税"转变为"事前规划"，在做决策前就考虑税务影响

本练习体系分为五个阶段，从最基础的税制认知到高级的综合筹划，每个阶段都有明确的目标、具体的练习任务和可量化的验收标准。

## 第一阶段：税制认知训练（第1-2周）

> 目标：建立完整的中国税制知识框架，掌握最常用的个税计算方法。

### 练习1：个人所得税计算实操

**目标**：掌握综合所得个人所得税的完整计算流程，能够独立计算自己的年度个税。

**为什么重要**：个税是大多数人唯一直接缴纳的税种。只有真正理解自己的个税是怎么算出来的，才能找到合法节税的切入点。很多人连自己每月交了多少税都不清楚，更不用说优化了。

**练习步骤**：

**第一步：采集基础数据**

从工资条或个税APP中获取以下信息：
- 每月应发工资（税前）
- 五险一金个人缴纳部分（养老8%、医疗2%、失业0.5%、公积金5%-12%）
- 是否有其他收入来源（劳务报酬、稿酬、特许权使用费）

**第二步：逐月计算预扣预缴税额**

```text
每月应纳税所得额 = 月收入 - 5,000（基本减除） - 专项扣除 - 专项附加扣除

预扣税率表（累计预扣法）：
┌────────────────────────────┬──────────┬──────────────┐
│ 累计应纳税所得额            │ 税率     │ 速算扣除数   │
├────────────────────────────┼──────────┼──────────────┤
│ 不超过36,000元              │ 3%       │ 0            │
│ 36,000 - 144,000元          │ 10%      │ 2,520        │
│ 144,000 - 300,000元         │ 20%      │ 16,920       │
│ 300,000 - 420,000元         │ 25%      │ 31,920       │
│ 420,000 - 660,000元         │ 30%      │ 52,920       │
│ 660,000 - 960,000元         │ 35%      │ 85,920       │
│ 超过960,000元               │ 45%      │ 181,920      │
└────────────────────────────┴──────────┴──────────────┘
```

**实操计算模板**：

```text

     预扣税额 = _________ × 3% = _________元

2月：累计所得额 = 上月累计 + 本月 = _________元
     累计预扣税额 = _________ × 3% = _________元

...（逐月计算，注意税率跳档时的税额变化）

12月汇总：
- 全年应发工资：_________元
- 全年专项扣除：_________元
- 全年专项附加扣除：_________元
- 全年应纳税所得额：_________元
- 全年应纳税额：_________元
```

### 练习2：专项附加扣除全面梳理

**目标**：确保自己充分利用了所有适用的专项附加扣除，不遗漏任何一项。

**为什么重要**：专项附加扣除是国家给纳税人的"真金白银"优惠。很多符合条件的人因为不了解政策而白白多交税。以一个有1个孩子、赡养老人、有房贷的纳税人为例，仅这三项每月就能扣除4,000元，一年少交税约1,440-3,600元（取决于税率档位）。

**练习步骤**：

**第一步：逐项排查**

对以下7个扣除项目，逐一确认自己是否符合条件：

```text
┌─────────────────────┬──────────────────────────────────┬────────┬──────┐
│ 扣除项目             │ 标准条件                           │ 是否符合│ 金额 │
├─────────────────────┼──────────────────────────────────┼────────┼──────┤
│ 子女教育             │ 3岁-博士阶段的子女                 │ □是□否 │____元│
│ 继续教育             │ 在读学历教育 或 取得职业资格证书    │ □是□否 │____元│
│ 大病医疗             │ 医保目录内自付累计超15,000元        │ □是□否 │____元│
│ 住房贷款利息         │ 首套住房贷款，还款期内              │ □是□否 │____元│
│ 住房租金             │ 在主要工作城市无自有住房             │ □是□否 │____元│
│ 赡养老人             │ 父母年满60岁                       │ □是□否 │____元│
│ 婴幼儿照护           │ 3岁以下婴幼儿                      │ □是□否 │____元│
└─────────────────────┴──────────────────────────────────┴────────┴──────┘

注意：住房贷款利息和住房租金不能同时享受，需选择更有利的一项。
```

**第二步：夫妻最优分配方案**

对于可以由夫妻一方扣除或双方分摊的项目，设计最优方案：

```text
方案设计原则：
1. 谁的税率高，扣除就给谁（边际税率高的人享受扣除，节税效果更大）
2. 如果双方税率相同，谁的收入更稳定就给谁
3. 大病医疗只能由本人扣除，或本人选择由配偶扣除

你的家庭情况：
- 本人税率档位：_______  配偶税率档位：_______

最优分配方案：
┌─────────────────────┬────────────────┬────────────────┐
│ 扣除项目             │ 扣除方          │ 年度节税金额    │
├─────────────────────┼────────────────┼────────────────┤
│ 子女教育             │                │                │
│ 住房贷款利息         │                │                │
│ 赡养老人             │                │                │
│ 婴幼儿照护           │                │                │
└─────────────────────┴────────────────┴────────────────┘

年度合计节税：_________元
```

**第三步：遗漏项补申报**

如果发现有符合条件但未申报的项目，立即在个人所得税APP上操作：
1. 打开个人所得税APP → 办税 → 专项附加扣除填报
2. 选择对应的扣除项目
3. 填写相关信息并提交
4. 选择通过扣缴义务人申报（当月生效）或综合所得年度自行申报

**验收标准**：确认已申报所有适用的专项附加扣除，完成夫妻最优分配方案设计，计算出年度节税总额。

### 练习3：年终奖计税方案选择

**目标**：掌握年终奖单独计税与并入综合所得两种方式的计算方法，能够判断哪种方式更优。

**为什么重要**：年终奖计税方式的选择直接影响到手金额。选错方式可能多交几千甚至上万元的税。2027年12月31日前，居民个人可以选择年终奖单独计税或并入综合所得，这个选择需要每年重新评估。

**核心知识点**：

```text
年终奖单独计税：
  应纳税额 = 年终奖 × 适用税率 - 速算扣除数
  其中适用税率 = 按年终奖÷12的商确定

  ┌────────────────────┬──────────┬──────────────┐
  │ 月均奖金（÷12）     │ 税率     │ 速算扣除数   │
  ├────────────────────┼──────────┼──────────────┤
  │ ≤3,000              │ 3%       │ 0            │
  │ 3,000-12,000        │ 10%      │ 210          │
  │ 12,000-25,000       │ 20%      │ 1,410        │
  │ 25,000-35,000       │ 25%      │ 2,660        │
  │ 35,000-55,000       │ 30%      │ 4,410        │
  │ 55,000-80,000       │ 35%      │ 7,160        │
  │ >80,000             │ 45%      │ 15,160       │
  └────────────────────┴──────────┴──────────────┘
```

**陷阱区间**（年终奖"多发少得"区间）：

```text
以下区间存在"多1元多交几千税"的陷阱：

36,001 - 38,567元：多发1元，多交2,310元税
144,001 - 160,500元：多发1元，多交13,200元税
300,001 - 318,333元：多发1元，多交16,000元税
420,001 - 447,500元：多发1元，多交21,000元税
660,001 - 706,538元：多发1元，多交30,000元税
960,001 - 1,120,000元：多发1元，多交45,000元税

应对策略：如果年终奖刚好落在陷阱区间，应将奖金降至区间起点以下，
差额以其他形式发放（如月薪调整、福利等）。
```

**练习场景**：

```text
场景A：小王月薪15,000元，五险一金3,000元，专项附加扣除2,000元，
      年终奖36,000元。请计算两种方式的税额，选择最优方案。

场景B：小李月薪25,000元，五险一金5,000元，专项附加扣除3,000元，
      年终奖150,000元。请计算两种方式的税额，选择最优方案。

场景C：小张月薪8,000元，五险一金1,500元，专项附加扣除1,000元，
      年终奖40,000元。注意：40,000在36,001-38,567的陷阱区间之外，
      但仍需比较两种方式。请计算并选择最优方案。
```

**计算模板**：

```text
场景___：

方式一：年终奖单独计税
- 工资部分年应纳税所得额 = (月薪 - 5,000 - 五险一金 - 专项附加扣除) × 12 = _________
- 工资部分个税 = _________（按综合所得税率表计算）
- 年终奖个税 = 年终奖 × 对应税率 - 速算扣除数 = _________
- 合计个税 = _________

方式二：年终奖并入综合所得
- 全年综合所得 = 工资收入 + 年终奖 = _________
- 全年应纳税所得额 = 综合所得 - 60,000 - 专项扣除 - 专项附加扣除 = _________
- 全年个税 = _________

结论：选择_______方式，节税_______元
```

**验收标准**：能够独立完成三种场景的计算，准确识别陷阱区间，给出正确的最优方案建议。

### 练习4：个税APP实操演练

**目标**：熟悉个人所得税APP的各项功能，掌握汇算清缴的操作流程。

**为什么重要**：个税APP是每个纳税人必须掌握的工具。每年3-6月的汇算清缴直接影响你的退税或补税金额。很多人因为不会操作而错过退税，或者因为操作错误而被要求补税。

**练习任务**：

```text
任务清单：
1. 下载并登录个人所得税APP（实名认证）
2. 查看自己的收入纳税明细
   - 路径：首页 → 收入纳税明细查询 → 选择年度
   - 核实：每笔收入是否正确，扣缴义务人是否正确
   - 异常处理：如发现"被就业"（从未任职的公司出现你的收入记录），
     立即通过APP申诉

3. 办理年度汇算
   - 路径：首页 → 综合所得年度汇算
   - 确认收入数据 → 确认扣除项目 → 计算应退税/补税
   - 退税：绑定银行卡，提交退税申请
   - 补税：年度综合所得收入不超12万 或 补税金额不超400元 可免补

4. 查询已申报的专项附加扣除
   - 路径：办税 → 专项附加扣除信息查询
   - 核实：所有已申报项目是否正确、完整

5. 办理其他业务（按需）
   - 经营所得申报（个体户/自由职业者）
   - 财产租赁所得申报
   - 个人养老金扣除管理
```

**验收标准**：能够独立完成个税APP的各项基本操作，特别是收入明细核对和汇算清缴流程。

## 第二阶段：个人投资税务优化（第3-6周）

> 目标：掌握各类个人投资涉及的税收，设计税务效率最优的投资方案。

### 练习5：股票投资税务全景分析

**目标**：全面了解A股个人投资者涉及的所有税种，理解不同持有策略的税务影响。

**为什么重要**：很多投资者只关注收益率，忽略税负对最终收益的侵蚀。长期持有和短期交易的税负差异巨大，理解这些差异是制定投资策略的基础。

**A股个人投资者税负全景**：

```text
┌──────────────┬───────────────────────────────────────────────────┐
│ 税种/费种     │ 规则                                             │
├──────────────┼───────────────────────────────────────────────────┤
│ 印花税        │ 卖出时按成交金额的0.05%征收（2023年8月起减半）       │
│ 股息红利税    │ 持有≤1个月：20%                                   │
│              │ 持有1个月-1年：10%                                 │
│              │ 持有>1年：免税                                     │
│ 资本利得税    │ 个人买卖A股股票免征个人所得税                       │
│ 佣金         │ 券商收取，一般万1-万3                              │
│ 过户费       │ 按成交金额的0.001%收取                             │
└──────────────┴───────────────────────────────────────────────────┘
```

**练习场景**：

```text
场景：你持有某股票10,000股，买入价10元/股，持有期间获得分红
     每股0.5元（即分红5,000元），之后以15元/股卖出。

请分别计算以下三种持有期限下的总税负：
1. 持有20天卖出
2. 持有6个月卖出
3. 持有18个月卖出

计算模板：
┌─────────────┬──────────┬──────────┬──────────┐
│ 项目         │ 持有20天  │ 持有6月   │ 持有18月  │
├─────────────┼──────────┼──────────┼──────────┤
│ 买入金额     │          │          │          │
│ 分红金额     │          │          │          │
│ 卖出金额     │          │          │          │
│ 印花税(卖)   │          │          │          │
│ 股息红利税   │          │          │          │
│ 资本利得税   │          │          │          │
│ 佣金(双向)   │          │          │          │

│ 总税费       │          │          │          │
│ 净收益       │          │          │          │
│ 税费占收益比 │          │          │          │
└─────────────┴──────────┴──────────┴──────────┘

结论：持有期限对税负的影响为_______
```

**进阶思考**：

| 问题 | 思考方向 |
|------|----------|
| 为什么A股不征收资本利得税？ | 政策目的：鼓励个人投资者参与资本市场 |

**验收标准**：能够独立计算不同持有期限下的投资总税负，理解长期持有的税务优势。

### 练习6：房产交易税费全流程计算

**目标**：掌握房产买卖各环节涉及的全部税费，能够预估房产交易的真实成本。

**为什么重要**：房产是大多数家庭最大的一笔交易，涉及的税费种类多、金额大。一套500万的住房，买卖过程中的税费可能高达几十万元。了解税费规则可以帮助你选择最优的交易时机和方式。

**房产交易税费全景**：

```text
买方税费：
┌───────────────────────────────────────────────────────────────┐
│ 契税                                                            │
│   首套房 ≤90㎡：成交价 × 1%                                     │
│   首套房 >90㎡：成交价 × 1.5%                                   │
│   二套房：成交价 × 1%（≤90㎡）或 2%（>90㎡）                    │
│   三套及以上：成交价 × 3%（部分城市）                            │
│                                                                 │
│ 其他费用：                                                      │
│   交易手续费：约3-6元/㎡                                        │
│   登记费：80元/套                                               │
│   权证印花税：5元/本                                            │
│   中介费：成交价 × 1%-3%（可协商）                              │
└───────────────────────────────────────────────────────────────┘

卖方税费：
┌───────────────────────────────────────────────────────────────┐
│ 增值税及附加                                                    │
```

### 练习8：小微企业优惠政策应用

**目标**：深入理解小微企业的认定标准和税收优惠政策，能够计算不同规模企业的最优纳税方案。

**为什么重要**：小微企业是中国经济的主体，国家对小微企业有力度很大的税收优惠。很多企业主不了解这些政策，或者不知道如何合法地享受优惠，导致多交了本可以节省的税。

**小微企业认定标准**（2023年起适用）：

```text
同时满足以下三个条件：
1. 年度应纳税所得额 ≤ 300万元
2. 从业人数 ≤ 300人
3. 资产总额 ≤ 5,000万元

优惠税率：
- 应纳税所得额 ≤ 300万元：
  减按25%计入应纳税所得额，按20%税率缴纳
  实际税负 = 25% × 20% = 5%
```

```text
3. 节税金额：_______元
```

### 练习9：研发费用加计扣除计算

**目标**：掌握研发费用加计扣除的政策要点和计算方法，能够评估企业申请高新技术企业的收益。

**为什么重要**：研发费用加计扣除是国家鼓励企业创新的核心政策。2023年起，所有企业的研发费用加计扣除比例统一为100%（集成电路和工业母机企业为120%），这意味着每投入100万元研发费用，可以在税前扣除200万元（或220万元），节税效果显著。

**政策要点**：

```text
研发费用加计扣除比例：
- 一般企业：100%（2023年起）
- 集成电路企业：120%
- 工业母机企业：120%

可加计扣除的研发费用范围：
1. 人员人工费用：研发人员工资、社保、公积金
2. 直接投入费用：材料、燃料、动力费用
3. 折旧费用：用于研发的仪器设备折旧
4. 无形资产摊销：用于研发的软件、专利等摊销
5. 设计试验费：新产品设计费、临床试验费等
6. 其他相关费用：差旅费、会议费等（限额10%）
7. 委托研发费用：按实际发生额的80%计入

不可加计扣除的费用：
- 常规性升级或改进
- 直接应用已有科研成果
- 商品化后的技术支持活动
- 重复或简单改变材料/产品
```

```text
   应纳税所得额 = 400 - 365 × 100% = _______万元
```

### 练习11：跨境税务筹划分析

**目标**：了解跨境税务的基本框架，能够分析常见跨境场景的税务影响。

**为什么重要**：随着全球化深入，越来越多的人有跨境收入或海外资产。不了解跨境税务规则可能导致双重征税，也可能因为不了解CRS而产生税务风险。

**核心知识点**：

```text
税务居民身份判定：
- 在中国有住所：全球征税
- 在中国无住所但居住满183天：中国境内所得 + 符合条件的境外所得
- 税收抵免：已在境外缴纳的所得税，可在国内税额中抵免（限额内）

CRS（共同申报准则）：
- 中国已与100+个国家/地区交换金融账户信息
- 交换内容：账户余额、利息、股息、出售金融资产的收入等
- 影响人群：在海外有金融账户的中国税务居民

┌────────────────────┬──────────────────────────────────────────────┐
```

### 练习12：税务风险防范案例分析

**目标**：识别常见的税务风险点，建立税务风险防范意识。

**为什么重要**：税务筹划的前提是合法合规。一旦触碰红线，面临的不仅是补税，还有滞纳金（每日万分之五）、罚款（0.5-5倍），甚至刑事责任。了解红线在哪里，才能在安全区内进行筹划。

**常见税务风险场景**：

```text
风险等级评估：

高风险（可能涉及刑事责任）：
┌─────────────────────────────────────────────────────────────┐
│ 1. 虚开发票：为他人、为自己、让他人为自己、介绍他人虚开        │
│    量刑标准：虚开税额5万元以上即可立案，50万以上可判10年以上     │
│                                                             │
│ 2. 隐匿收入：通过私人账户收款不入账、不申报                    │
│    金税四期下，个人账户大额交易已被重点监控                     │
│                                                             │
│ 3. 买卖发票：购买发票用于抵扣或列支成本                        │
│    虚开增值税专用发票：最高可判无期徒刑                         │
└─────────────────────────────────────────────────────────────┘

中风险（行政处罚为主）：
┌─────────────────────────────────────────────────────────────┐
│ 4. 关联交易转移利润：价格不公允，被纳税调整                    │
│    调整方法：可比非受控价格法、再销售价格法等                   │
│                                                             │
│ 5. 滥用核定征收：业务实质不符，享受核定征收优惠                │
│    风险：政策收紧，可能被追缴往年税款                          │
│                                                             │
│ 6. 不合规的业务拆分：拆分无真实商业目的，仅为避税              │
│    后果：被合并征税 + 滞纳金 + 罚款                           │
└─────────────────────────────────────────────────────────────┘

低风险（需要关注但后果较轻）：
┌─────────────────────────────────────────────────────────────┐
│ 7. 申报错误：计算错误、遗漏收入等非故意行为                    │
│    处理：补税 + 滞纳金，一般不罚款                             │
│                                                             │
│ 8. 政策理解偏差：对税收优惠政策理解不准确                      │
│    处理：补税 + 滞纳金，主动更正一般不罚款                     │
└─────────────────────────────────────────────────────────────┘
```

**案例分析练习**：

```text
案例1：个人账户收款
张某经营一家餐饮店，年营收约200万元。为了少交税，张某让顾客
扫码支付到个人微信账户，不入账不申报。三年累计隐匿收入约500万元。

请分析：
1. 涉及哪些违法行为？_______
2. 可能面临的补税金额（假设综合税负率10%）：_______万元
3. 滞纳金（按每日万分之五，假设延迟3年）：_______万元
4. 罚款倍数（假设按1倍）：_______万元
5. 是否可能涉及刑事责任？_______

案例2：核定征收滥用
李某在税收洼地注册了一家个人独资企业（核定征收），然后以该
企业的名义与自己的有限公司签订咨询服务合同，每年开具200万
元咨询费发票。实际上并没有真实的咨询服务。

请分析：
1. 这种行为的税务风险是什么？_______

案例3：股权代持
王某让朋友赵某代持一家公司的股权。公司分红100万元，赵某收到
后转给王某。赵某未就该分红申报个税。

请分析：
1. 纳税义务人是谁？_______

```

### 练习13：年度税务健康检查

**目标**：建立系统化的税务健康检查流程，每年对自己的税务状况进行全面审查。

**为什么重要**：税务筹划不是一次性的事情，而是需要持续关注和调整的动态过程。每年进行一次税务健康检查，可以及时发现遗漏的优惠、潜在的风险和优化的空间。

**税务健康检查清单**：

```text
一、个人税务检查（建议每年1月进行）

□ 1. 收入完整性
   - 工资薪金：核对工资条与个税APP记录是否一致
   - 劳务报酬：确认所有代扣代缴是否正确
   - 投资收益：股票分红、基金分红、利息收入
   - 其他收入：稿酬、特许权使用费、偶然所得

□ 2. 扣除充分性
   - 专项附加扣除：是否所有适用项目已申报
   - 扣除金额是否准确（如子女是否毕业、老人是否健在等）
   - 夫妻扣除分配是否最优
   - 个人养老金是否已缴满12,000元

□ 3. 优惠政策利用
   - 年终奖计税方式是否选择了最优方案
   - 是否有符合条件的税收优惠未享受
   - 公益捐赠是否取得了税前扣除凭证

□ 4. 投资组合税务效率
   - 是否持有不必要的高税负产品
   - 亏损投资是否考虑了税务角度的处置时机
   - 是否有跨账户（普通账户vs养老金账户）的优化空间

□ 5. 风险排查
   - 是否有未申报的海外收入或资产
   - 是否有被代持的股权需要处理
   - 是否有需要补缴税款的情况

二、企业税务检查（建议每季度进行）

□ 1. 发票管理
   - 进项发票是否及时取得和认证
   - 销项发票是否按规定开具
   - 是否存在虚开发票的风险

□ 2. 成本费用合规性
   - 所有支出是否有合规凭证
   - 个人费用与企业费用是否混同
   - 关联交易价格是否公允

□ 3. 优惠政策利用
   - 是否满足小微企业条件并享受优惠
   - 研发费用是否已归集并享受加计扣除
   - 是否有其他适用的税收优惠未享受

□ 4. 现金流管理
   - 税款缴纳时间是否合理安排
   - 是否有可申请延期缴纳的情况
   - 退税申请是否及时提交
```

**练习任务**：

```text
请按照上述清单，对自己的税务状况进行一次全面检查：

个人检查结果：
- 已检查项目数：_______
- 发现问题数：_______
- 潜在节税空间：_______元/年
- 需要补申报的项目：_______
- 风险等级：□低 □中 □高

下一步行动计划：
1. _______
2. _______
3. _______
```

**验收标准**：完成一次完整的税务健康检查，识别出至少一个可优化的点。
