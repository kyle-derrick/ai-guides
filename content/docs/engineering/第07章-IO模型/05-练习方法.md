# IO模型练习方法

本章的练习方法围绕Linux五大IO模型（阻塞IO、非阻塞IO、IO多路复用、信号驱动IO、异步IO）设计，从概念理解到系统编程实战，从单机诊断到高并发架构，覆盖从初级到资深工程师的完整能力谱系。每个练习都配有明确的目标、步骤、预期输出和自检标准。

---

## 练习一：IO模型核心概念（预计45分钟）

### 目标

能够准确区分五种IO模型的行为差异，理解内核空间与用户空间的数据流转，能手绘IO操作的两阶段模型并标注阻塞/非阻塞、同步/异步的关键区别点。

### 步骤

**1. 复习IO操作的两阶段模型（10分钟）**

回顾Stevens在《UNIX网络编程》中提出的经典模型，写出以下表格的完整版本：

| 阶段       | 描述                           | 阻塞IO行为          | 非阻塞IO行为           |
|-----------|-------------------------------|--------------------|-----------------------|
| 数据准备   | 内核等待数据从硬件到达缓冲区     | 进程阻塞直到数据到达  | 内核立即返回EAGAIN      |
| 数据拷贝   | 内核缓冲区→用户缓冲区           | 进程继续阻塞完成拷贝  | 内核执行拷贝后立即返回   |

**关键思考题**：为什么"非阻塞"只改变了阶段一的行为，阶段二的数据拷贝仍然需要内核参与？

**2. 手绘五种IO模型的时序图（15分钟）**

不参考任何资料，凭记忆画出五种模型的时序图。以`read()`系统调用为例，标注以下要素：

| 要素                | 需要标注的内容                                    |
|--------------------|------------------------------------------------|
| 用户进程状态         | 运行、阻塞、空闲轮询                              |
| 系统调用返回时机      | 数据准备好前/后返回                               |
| 数据拷贝方式         | 同步拷贝（阶段二始终同步）vs 异步通知              |
| 额外机制            | select/poll/epoll fd、信号handler、aio回调        |

参考答案对照（画完后核对）：

- **阻塞IO**：read()发起→进程挂起→数据到达→内核拷贝→read()返回。全程阻塞。
- **非阻塞IO**：read()返回EAGAIN→进程做其他事→轮询read()→数据到达→拷贝完成→返回。需要应用层轮询。
- **IO多路复用**：select/epoll_wait()阻塞→某个fd就绪→read()执行→拷贝完成→返回。一次等待多个fd。
- **信号驱动IO**：注册SIGIO handler→read()立即返回→内核数据到达时发SIGIO→handler中read()→拷贝完成。内核主动通知。
- **异步IO**：aio_read()提交请求→立即返回→内核完成拷贝后发通知（信号/回调/线程）→应用处理结果。全程不阻塞。

**3. 同步vs异步的本质辨析（10分钟）**

用以下测试检验自己的理解：对于每种模型，回答两个问题：
- "阶段一（数据准备）时进程能否做其他工作？"
- "阶段二（数据拷贝）是谁发起的？"

| 模型           | 阶段一能否做其他工作 | 谁发起数据拷贝  | 分类     |
|---------------|-------------------|--------------|---------|
| 阻塞IO         | 否（阻塞）         | 内核（进程等待）| 同步IO   |
| 非阻塞IO       | 是（轮询间隙）      | 内核（进程触发）| 同步IO   |
| IO多路复用      | 否（select阻塞）   | 内核（进程触发）| 同步IO   |
| 信号驱动IO      | 是（异步等待信号）   | 内核（进程触发）| 同步IO   |
| 异步IO         | 是（完全不阻塞）     | 内核（自动完成）| 异步IO   |

**常见误区**：很多人把"非阻塞IO"误认为是"异步IO"。关键区别在于非阻塞IO的数据拷贝阶段仍然由进程发起，而异步IO的整个过程（包括拷贝）都由内核完成。

**4. 对比框架总结（10分钟）**

完成以下对比框架的填空，作为长期记忆锚点：

| 维度           | 阻塞IO | 非阻塞IO | IO多路复用 | 信号驱动IO | 异步IO |
|---------------|--------|---------|----------|-----------|-------|
| 并发能力       | 极低    | 低       | 高       | 中高       | 最高   |
| CPU利用率      | 低      | 中       | 高       | 高         | 最高   |
| 编程复杂度     | 简单    | 中等     | 较高     | 高         | 最高   |
| 实际使用场景   | 简单脚本| 专用协议 | Web服务器 | 特殊场景   | 高性能  |
| Linux实现机制  | 默认行为 | O_NONBLOCK| select/poll/epoll | SIGIO | AIO/io_uring |

### 检查标准

- [ ] 能不借助资料画出五种IO模型的完整时序图
- [ ] 能清晰解释"同步IO"和"异步IO"的本质区别（数据拷贝由谁发起）
- [ ] 能说出每种模型至少一个真实应用场景
- [ ] 能正确判断一个IO场景属于哪种模型

## 练习二：用C语言实现五种IO模型（预计120分钟）

### 目标

亲手编写并运行每种IO模型的echo server，从系统调用层面理解五种模型的行为差异。这是本章最核心的实操练习。

### 步骤

**1. 阻塞IO echo server（20分钟）**

创建 `blocking_server.c`：

```c
#include <stdio.h>
#include <stdlib.h>
#include <string.h>
#include <unistd.h>
#include <arpa/inet.h>

#define PORT 8801
#define BUF_SIZE 1024

int main() {
    int server_fd, client_fd;
    struct sockaddr_in addr;
    char buf[BUF_SIZE];

    // 创建TCP socket
    server_fd = socket(AF_INET, SOCK_STREAM, 0);
    int opt = 1;
    setsockopt(server_fd, SOL_SOCKET, SO_REUSEADDR, &amp;opt, sizeof(opt));

    addr.sin_family = AF_INET;
    addr.sin_addr.s_addr = INADDR_ANY;
    addr.sin_port = htons(PORT);
    bind(server_fd, (struct sockaddr*)&amp;addr, sizeof(addr));
    listen(server_fd, 128);

    printf("[阻塞IO Server] 监听端口 %d，等待连接...\n", PORT);

    while (1) {
        // accept是阻塞调用——没有客户端连接时，进程挂起
        client_fd = accept(server_fd, NULL, NULL);
        printf("[新连接] fd=%d\n", client_fd);

        // 每次只处理一个客户端（单线程阻塞模型的局限性）
        while (1) {
            // read是阻塞调用——没有数据时，进程挂起
            ssize_t n = read(client_fd, buf, BUF_SIZE);
            if (n <= 0) {
                printf("[断开] fd=%d\n", client_fd);
                close(client_fd);
                break;
            }
            printf("[收到] fd=%d, %zd字节: %.*s", client_fd, n, (int)n, buf);
            write(client_fd, buf, n);  // echo回去
        }
    }

    close(server_fd);
    return 0;
}
```

```bash
gcc -o blocking_server blocking_server.c
./blocking_server &amp;

# 用netcat测试
echo "hello blocking IO" | nc localhost 8801
```

**观察重点**：服务器只能同时处理一个客户端。第二个客户端连接时，必须等第一个客户端断开才能被accept。

**2. 非阻塞IO echo server（20分钟）**

创建 `nonblocking_server.c`：

```c
#include <stdio.h>
#include <stdlib.h>
#include <string.h>
#include <unistd.h>
#include <fcntl.h>
#include <errno.h>
#include <arpa/inet.h>

#define PORT 8802
#define BUF_SIZE 1024
#define MAX_CLIENTS 128

int main() {
    int server_fd;
    struct sockaddr_in addr;

    server_fd = socket(AF_INET, SOCK_STREAM, 0);
    int opt = 1;
    setsockopt(server_fd, SOL_SOCKET, SO_REUSEADDR, &amp;opt, sizeof(opt));

    // 关键：设置非阻塞模式
    fcntl(server_fd, F_SETFL, O_NONBLOCK);

    addr.sin_family = AF_INET;
    addr.sin_addr.s_addr = INADDR_ANY;
    addr.sin_port = htons(PORT);
    bind(server_fd, (struct sockaddr*)&amp;addr, sizeof(addr));
    listen(server_fd, 128);

    int clients[MAX_CLIENTS] = {0};
    int client_count = 0;
    char buf[BUF_SIZE];

    printf("[非阻塞IO Server] 监听端口 %d\n", PORT);

    while (1) {
        // 非阻塞accept——没有新连接时立即返回-1, errno=EAGAIN
        int new_fd = accept(server_fd, NULL, NULL);
        if (new_fd != -1) {
            fcntl(new_fd, F_SETFL, O_NONBLOCK);  // 客户端fd也设为非阻塞
            if (client_count < MAX_CLIENTS) {
                clients[client_count++] = new_fd;
                printf("[新连接] fd=%d, 当前%d个客户端\n", new_fd, client_count);
            }
        }

        // 轮询所有已连接客户端——这就是非阻塞IO的核心模式
        for (int i = 0; i < client_count; i++) {
            ssize_t n = read(clients[i], buf, BUF_SIZE);
            if (n > 0) {
                printf("[收到] fd=%d, %zd字节\n", clients[i], n);
                write(clients[i], buf, n);
            } else if (n == 0) {
                // 客户端断开
                printf("[断开] fd=%d\n", clients[i]);
                close(clients[i]);
                clients[i] = clients[--client_count];  // 用最后一个填补空位
                i--;  // 重新检查当前位置
            } else if (errno != EAGAIN &amp;&amp; errno != EWOULDBLOCK) {
                // 真正的错误
                perror("read");
                close(clients[i]);
                clients[i] = clients[--client_count];
                i--;
            }
            // n == -1 &amp;&amp; errno == EAGAIN：没有数据，继续轮询下一个
        }

        // 注意：这里没有usleep！纯忙轮询，CPU占用100%
        // 生产环境需要加sleep或用select/epoll替代
    }

    close(server_fd);
    return 0;
}
```

```bash
gcc -o nonblocking_server nonblocking_server.c
./nonblocking_server &amp;

# 多个客户端同时连接测试
echo "client1" | nc localhost 8802 &amp;
echo "client2" | nc localhost 8802 &amp;
```

**观察重点**：服务器可以同时处理多个客户端了！但注意CPU占用——非阻塞IO的忙轮询模式会吃满一个CPU核心。

```bash
# 你会看到CPU%接近100%——这就是忙轮询的代价
```

**3. IO多路复用 — select版本（20分钟）**

创建 `select_server.c`：

```c
#include <stdio.h>
#include <stdlib.h>
#include <string.h>
#include <unistd.h>
#include <arpa/inet.h>

#define PORT 8803
#define BUF_SIZE 1024
#define MAX_CLIENTS 1024

int main() {
    int server_fd;
    struct sockaddr_in addr;
    char buf[BUF_SIZE];

    server_fd = socket(AF_INET, SOCK_STREAM, 0);
    int opt = 1;
    setsockopt(server_fd, SOL_SOCKET, SO_REUSEADDR, &amp;opt, sizeof(opt));

    addr.sin_family = AF_INET;
    addr.sin_addr.s_addr = INADDR_ANY;
    addr.sin_port = htons(PORT);
    bind(server_fd, (struct sockaddr*)&amp;addr, sizeof(addr));
    listen(server_fd, 128);

    fd_set read_fds;
    int max_fd = server_fd;
    int clients[MAX_CLIENTS] = {0};

    printf("[Select Server] 监听端口 %d\n", PORT);

    while (1) {
        FD_ZERO(&amp;read_fds);
        FD_SET(server_fd, &amp;read_fds);
        for (int i = 0; i < MAX_CLIENTS; i++) {
            if (clients[i] > 0) {
                FD_SET(clients[i], &amp;read_fds);
                if (clients[i] > max_fd) max_fd = clients[i];
            }
        }

        // select阻塞等待，直到至少一个fd就绪
        int activity = select(max_fd + 1, &amp;read_fds, NULL, NULL, NULL);
        if (activity < 0) { perror("select"); continue; }

        // 检查是否有新连接
        if (FD_ISSET(server_fd, &amp;read_fds)) {
            int new_fd = accept(server_fd, NULL, NULL);
            if (new_fd != -1) {
                for (int i = 0; i < MAX_CLIENTS; i++) {
                    if (clients[i] == 0) {
                        clients[i] = new_fd;
                        printf("[新连接] fd=%d, slot=%d\n", new_fd, i);
                        break;
                    }
                }
            }
        }

        // 检查已连接客户端的数据
        for (int i = 0; i < MAX_CLIENTS; i++) {
            if (clients[i] > 0 &amp;&amp; FD_ISSET(clients[i], &amp;read_fds)) {
                ssize_t n = read(clients[i], buf, BUF_SIZE);
                if (n <= 0) {
                    printf("[断开] fd=%d\n", clients[i]);
                    close(clients[i]);
                    clients[i] = 0;
                } else {
                    write(clients[i], buf, n);
                }
            }
        }
    }

    close(server_fd);
    return 0;
}
```

```bash
gcc -o select_server select_server.c
./select_server &amp;
```

**观察重点**：
- CPU占用大幅下降（不再忙轮询）
- select有FD_SETSIZE限制（通常1024），不适合超大规模连接
- select每次调用都需要把fd集合从用户空间拷贝到内核空间

**4. IO多路复用 — epoll版本（20分钟）**

创建 `epoll_server.c`：

```c
#include <stdio.h>
#include <stdlib.h>
#include <string.h>
#include <unistd.h>
#include <arpa/inet.h>
#include <sys/epoll.h>

#define PORT 8804
#define BUF_SIZE 1024
#define MAX_EVENTS 64

int main() {
    int server_fd, epoll_fd;
    struct sockaddr_in addr;
    struct epoll_event ev, events[MAX_EVENTS];
    char buf[BUF_SIZE];

    server_fd = socket(AF_INET, SOCK_STREAM, 0);
    int opt = 1;
    setsockopt(server_fd, SOL_SOCKET, SO_REUSEADDR, &amp;opt, sizeof(opt));

    addr.sin_family = AF_INET;
    addr.sin_addr.s_addr = INADDR_ANY;
    addr.sin_port = htons(PORT);
    bind(server_fd, (struct sockaddr*)&amp;addr, sizeof(addr));
    listen(server_fd, 128);

    // 创建epoll实例
    epoll_fd = epoll_create1(0);

    // 注册server_fd的读事件
    ev.events = EPOLLIN;
    ev.data.fd = server_fd;
    epoll_ctl(epoll_fd, EPOLL_CTL_ADD, server_fd, &amp;ev);

    printf("[Epoll Server] 监听端口 %d\n", PORT);

    while (1) {
        // epoll_wait阻塞等待，只返回就绪的fd
        int nfds = epoll_wait(epoll_fd, events, MAX_EVENTS, -1);

        for (int i = 0; i < nfds; i++) {
            if (events[i].data.fd == server_fd) {
                int new_fd = accept(server_fd, NULL, NULL);
                if (new_fd != -1) {
                    ev.events = EPOLLIN;
                    ev.data.fd = new_fd;
                    epoll_ctl(epoll_fd, EPOLL_CTL_ADD, new_fd, &amp;ev);
                    printf("[新连接] fd=%d\n", new_fd);
                }
            } else {
                int client_fd = events[i].data.fd;
                ssize_t n = read(client_fd, buf, BUF_SIZE);
                if (n <= 0) {
                    printf("[断开] fd=%d\n", client_fd);
                    epoll_ctl(epoll_fd, EPOLL_CTL_DEL, client_fd, NULL);
                    close(client_fd);
                } else {
                    write(client_fd, buf, n);
                }
            }
        }
    }

    close(epoll_fd);
    close(server_fd);
    return 0;
}
```

**5. 用Python验证IO模型行为（20分钟）**

Python的`socket`模块可以方便地验证IO模型。创建 `io_model_demo.py`：

```python
import socket
import select
import time
import sys

def demo_blocking():
    """阻塞IO演示"""
    print("=== 阻塞IO演示 ===")
    s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
    s.bind(('127.0.0.1', 0))
    s.listen(1)
    addr = s.getsockname()
    print(f"监听 {addr}")

    # 客户端线程
    import threading
    def client():
        time.sleep(1)  # 故意延迟1秒
        c = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
        c.connect(addr)
        c.send(b'hello')
        c.close()

    threading.Thread(target=client, daemon=True).start()

    print("  调用accept()前，进程将阻塞...")
    start = time.time()
    conn, _ = s.accept()  # 阻塞调用
    elapsed = time.time() - start
    print(f"  accept()返回，耗时 {elapsed:.1f}秒 (应该≈1秒)")

    data = conn.recv(1024)
    print(f"  收到数据: {data}")
    conn.close()
    s.close()

def demo_nonblocking():
    """非阻塞IO演示"""
    print("\n=== 非阻塞IO演示 ===")
    s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
    s.setblocking(False)  # 关键：非阻塞
    s.bind(('127.0.0.1', 0))
    s.listen(1)
    addr = s.getsockname()

    import threading
    def client():
        time.sleep(1)
        c = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
        c.connect(addr)
        c.send(b'hello')
        c.close()

    threading.Thread(target=client, daemon=True).start()

    print("  尝试accept()，应该立即返回错误...")
    try:
        conn, _ = s.accept()
    except BlockingIOError as e:
        print(f"  accept()抛出 BlockingIOError: {e}")

    print("  开始轮询（每0.1秒一次）...")
    for attempt in range(30):  # 最多等3秒
        try:
            conn, _ = s.accept()
            print(f"  第{attempt+1}次轮询，accept成功!")
            data = conn.recv(1024)
            print(f"  收到数据: {data}")
            conn.close()
            break
        except BlockingIOError:
            print(f"  第{attempt+1}次轮询，无连接（EAGAIN），继续...")
            time.sleep(0.1)

    s.close()

def demo_multiplexing():
    """IO多路复用演示"""
    print("\n=== IO多路复用(select)演示 ===")
    s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
    s.bind(('127.0.0.1', 0))
    s.listen(128)
    addr = s.getsockname()

    import threading
    def client(n):
        time.sleep(0.5)
        c = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
        c.connect(addr)
        c.send(f'client{n}'.encode())
        time.sleep(2)  # 保持连接
        c.close()

    for i in range(3):
        threading.Thread(target=client, args=(i,), daemon=True).start()

    start = time.time()
    readable = [s]
    while readable:
        ready, _, _ = select.select(readable, [], [], 0.5)
        for sock in ready:
            if sock is s:
                conn, _ = s.accept()
                readable.append(conn)
                print(f"  新连接 fd={conn.fileno()}")
            else:
                data = sock.recv(1024)
                if data:
                    print(f"  收到 fd={sock.fileno()}: {data.decode()}")
                    sock.close()
                    readable.remove(sock)

    print(f"  总耗时 {time.time()-start:.1f}秒，处理了3个并发连接")
    s.close()

if __name__ == '__main__':
    demo_blocking()
    demo_nonblocking()
    demo_multiplexing()
```

- 非阻塞IO：accept立即抛出BlockingIOError，随后轮询成功
- IO多路复用：select同时监听多个fd，依次处理3个连接

**3. 连接池设计与测试（20分钟）**

```python
# 连接池对比实验
import socket
import time
import threading
from collections import deque

class SimpleConnectionPool:
    """简单连接池实现"""

    def __init__(self, host, port, max_size=20, min_idle=5):
        self.host = host
        self.port = port
        self.max_size = max_size
        self.pool = deque()
        self.lock = threading.Lock()
        self.current_size = 0

        # 预创建最小空闲连接
        for _ in range(min_idle):
            conn = self._create_connection()
            self.pool.append(conn)
            self.current_size += 1

    def _create_connection(self):
        s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
        s.connect((self.host, self.port))
        s.settimeout(5)
        return s

    def get(self, timeout=5):
        """获取一个连接"""
        with self.lock:
            if self.pool:
                return self.pool.popleft()
        # 池空了，创建新连接（不超过上限）
        if self.current_size < self.max_size:
            with self.lock:
                self.current_size += 1
            return self._create_connection()
        # 超过上限，等待归还
        # 生产环境应该用Condition实现等待
        raise Exception("连接池已满")

    def put(self, conn):
        """归还连接"""
        with self.lock:
            if len(self.pool) < self.max_size:
                self.pool.append(conn)
            else:
                conn.close()
                self.current_size -= 1

# 使用示例
pool = SimpleConnectionPool('127.0.0.1', 8804)

# 模拟并发请求
latencies = []

def make_request():
    start = time.time()
    try:
        conn = pool.get()
        conn.send(b'pool test')
        data = conn.recv(1024)
        pool.put(conn)
        latency = time.time() - start
        latencies.append(latency)
    except Exception as e:
        latencies.append(time.time() - start)

threads = [threading.Thread(target=make_request) for _ in range(100)]
for t in threads:
    t.start()
for t in threads:
    t.join()

import statistics
print(f"平均延迟: {statistics.mean(latencies)*1000:.1f}ms")
print(f"P99延迟: {sorted(latencies)[int(len(latencies)*0.99)]*1000:.1f}ms")
print(f"连接池大小: {len(pool.pool)} (空闲)")
```
