## 九、技术分析核心技巧

技术分析是通过研究市场过去的价格和成交量数据来预测未来价格走势的方法。与基本面分析关注"公司值多少钱"不同，技术分析回答的是"市场认为它值多少钱"以及"接下来市场会怎么走"。技术分析的三大假设是：市场行为包容消化一切信息、价格以趋势方式演变、历史会重演。

本章从K线语言、均线体系、成交量密码、核心技术指标、趋势与形态、支撑阻力、多周期分析到综合实战，系统覆盖技术分析的完整知识体系。

### 9.1 K线形态识别

K线（蜡烛图）起源于18世纪日本米市交易，由本间宗久系统化，后经Steve Nison引入西方。每根K线记录了四个价格：开盘价、最高价、最低价、收盘价，是市场多空博弈的直观呈现。

#### 9.1.1 单根K线的结构与含义

```text
阳线（收盘价>开盘价）：        阴线（收盘价<开盘价）：
  ┃                           ┃
  ┃  上影线                   ┃  上影线
  ┃                           ┃
 ┌┴┐                         ┌┴┐
 │ │  实体（红色/空心）        │ │  实体（绿色/实心）
 │ │                         │ │
 └┬┘                         └┬┘
  ┃                           ┃
  ┃  下影线                   ┃  下影线
  ┃                           ┃
```

实体大小反映多空力量的强弱程度。实体越长，一方力量越强。影线则反映盘中遭遇的抵抗——上影线越长说明上方卖压越重，下影线越长说明下方买盘越强。

**特殊单根K线形态详解**：

| 形态名称 | 特征 | 市场含义 | 出现位置 | 后续操作建议 |
|---------|------|---------|---------|-------------|
| 大阳线 | 实体长度>5%，几乎无影线 | 多方强势控盘，趋势加速 | 上涨初期=启动信号；上涨末期=赶顶 | 初期可追，末期警惕 |
| 大阴线 | 实体长度>5%，几乎无影线 | 空方强势控盘，抛压沉重 | 下跌初期=破位；下跌末期=恐慌性抛售 | 初期止损，末期观察是否放量 |
| 锤子线 | 下影线≥实体2倍，无上影线 | 下方买盘强劲，空方被反击 | 下跌趋势末端 | 次日收阳可轻仓试探 |
| 倒锤子线 | 上影线≥实体2倍，无下影线 | 上方试盘遇阻，但多方开始进攻 | 下跌趋势末端 | 需次日收阳确认 |
| 十字星 | 开盘价≈收盘价，上下影线可长可短 | 多空力量暂时均衡 | 任何位置 | 结合前后K线判断方向 |
| 长腿十字星 | 上下影线都很长 | 多空激烈争夺，市场犹豫不决 | 趋势末端=反转信号 | 等待后续确认 |
| 墓碑十字星 | 只有长上影线，无下影线和实体 | 上方被完全压制 | 上涨末端 | 强烈看跌信号 |
| T字线 | 只有长下影线，无上影线和实体 | 下方支撑极强 | 下跌末端 | 强烈看涨信号 |
| 一字线 | 四价合一 | 极端涨跌停，流动性枯竭 | 重大消息后 | 等待打开再操作 |

#### 9.1.2 经典K线组合形态

两根或多根K线的组合比单根K线更具判断价值，因为它们展示了多空力量的演变过程。

**底部反转组合**：

| 组合名称 | 构成 | 市场含义 | 可靠度 | 确认要素 |
|---------|------|---------|--------|---------|
| 早晨之星 | 大阴线→小实体(星线)→大阳线 | 空方力竭→犹豫→多方反攻 | ★★★★ | 第三根阳线实体越长越可靠 |
| 看涨吞没 | 阴线→阳线完全包住前阴线 | 空方被全面反包，力量逆转 | ★★★★ | 阳线成交量放大更佳 |
| 看涨刺透 | 大阴线→阳线开盘低于前低但收过前实体中点 | 空方创出新低后被强烈反击 | ★★★ | 阳线收盘超过前实体一半 |
| 红三兵 | 三根依次上升的小阳线 | 多方稳步推进，趋势健康 | ★★★ | 每根收盘价依次升高 |
| 平底 | 两根或多根K线最低价相同 | 同一价位多次获得支撑 | ★★ | 需后续放量上攻确认 |

**顶部反转组合**：

| 组合名称 | 构成 | 市场含义 | 可靠度 | 确认要素 |
|---------|------|---------|--------|---------|
| 黄昏之星 | 大阳线→小实体(星线)→大阴线 | 多方力竭→犹豫→空方反攻 | ★★★★ | 第三根阴线放量更可靠 |
| 看跌吞没 | 阳线→阴线完全包住前阳线 | 多方被全面反包，力量逆转 | ★★★★ | 阴线放量效果更强 |
| 乌云盖顶 | 大阳线→阴线开盘高于前高但收过前实体中点 | 上方抛压沉重 | ★★★ | 阴线成交量放大更可靠 |
| 三只乌鸦 | 三根依次下跌的阴线 | 空方持续打压，趋势转弱 | ★★★ | 每根收盘价依次降低 |
| 平顶 | 两根或多根K线最高价相同 | 同一价位多次遇阻 | ★★ | 需后续跌破确认 |

#### 9.1.3 K线分析的实战要点

**时间周期的选择**：日K线用于中短线交易，周K线用于中线趋势判断，月K线用于长期趋势确认。60分钟和15分钟K线用于日内交易的精确入场。不同周期的K线信号可能矛盾——周期越长的信号越可靠但越滞后，周期越短的信号越灵敏但噪音越多。实战中建议以日K为主、周K定方向、分钟K精确入场。

**K线分析的常见陷阱**：

- **孤立看单根K线**：一根锤子线出现在上涨中途可能只是普通回调，只有在下跌趋势末端才构成有意义的反转信号
- **忽略成交量配合**：看涨吞没形态如果没有成交量放大，可靠性大打折扣
- **过度解读小实体**：小幅波动中的十字星或小阴小阳不具有方向性意义
- **忽略市场环境**：牛市中阴线的意义与熊市完全不同，同样的形态在不同趋势中含义相反

### 9.2 均线系统

均线（Moving Average）是最基础也是最实用的趋势跟踪工具。其本质是对过去N个交易日收盘价的算术平均，通过"平滑"价格波动来揭示趋势方向。

#### 9.2.1 均线的类型与计算

| 类型 | 计算方式 | 特点 | 适用场景 |
|------|---------|------|---------|
| SMA（简单移动平均） | N日收盘价之和÷N | 计算简单，各日权重相等 | 最常用，适合趋势判断 |
| EMA（指数移动平均） | 给近期价格更高权重 | 对价格变化更敏感 | 适合短线交易信号 |
| WMA（加权移动平均） | 线性递减权重 | 介于SMA和EMA之间 | 特定策略使用 |
| VWAP（成交量加权平均） | 以成交量为权重的均价 | 反映真实成交成本 | 机构交易日内参考 |

#### 9.2.2 常用均线体系

| 均线 | 周期 | 含义 | 用途 |
|------|------|------|------|
| MA5 | 5日 | 短期攻击线 | 超短线操作参考，股价不破MA5可持有 |
| MA10 | 10日 | 短期操盘线 | 短线趋势确认，破位需减仓 |
| MA20 | 20日 | 中期生命线 | 波段操作核心参考，多空分水岭 |
| MA60 | 60日 | 季度趋势线 | 中线趋势判断，机构成本线 |
| MA120 | 120日 | 半年线（决策线） | 中长期趋势确认 |
| MA250 | 250日 | 年线（牛熊线） | 长期牛熊转换标志 |

#### 9.2.3 均线经典信号

**金叉与死叉**：

金叉是指短期均线从下方穿越长期均线，通常被视为买入信号；死叉则是短期均线从上方穿越长期均线，通常被视为卖出信号。但并非所有金叉都该买、所有死叉都该卖——零轴附近的金叉/死叉比高位/低位的更有参考意义。

```python
# 用Python计算金叉死叉信号
import pandas as pd

def ma_signals(df, short=5, long=20):
    """计算均线金叉死叉信号"""
    df['MA_short'] = df['close'].rolling(short).mean()
    df['MA_long'] = df['close'].rolling(long).mean()
    
    # 金叉：短期均线从下向上穿越长期均线
    df['golden_cross'] = (
        (df['MA_short'] > df['MA_long']) & 
        (df['MA_short'].shift(1) <= df['MA_long'].shift(1))
    )
    # 死叉：短期均线从上向下穿越长期均线
    df['death_cross'] = (
        (df['MA_short'] < df['MA_long']) & 
        (df['MA_short'].shift(1) >= df['MA_long'].shift(1))
    )
    return df
```

**均线排列**：

| 排列状态 | 均线形态 | 含义 | 操作策略 |
|---------|---------|------|---------|
| 多头排列 | MA5>MA10>MA20>MA60 | 强势上涨趋势 | 持有为主，回调到均线附近加仓 |
| 空头排列 | MA5<MA10<MA20<MA60 | 强势下跌趋势 | 空仓观望，反弹到均线附近减仓 |
| 交叉缠绕 | 各均线反复交叉 | 震荡盘整，无明确趋势 | 不参与或高抛低吸 |
| 粘合发散 | 多条均线粘合后分开 | 变盘信号，即将走出趋势 | 等待方向明确后跟进 |

**均线实战策略——葛兰碧八大法则**：

葛兰碧（Joseph Granville）提出的八大买卖法则是均线分析的经典框架：

```text
买入信号（四种）：
① 均线从下降转为走平或上升，价格从下方突破均线 → 趋势反转买入
② 价格虽跌破均线但均线仍上升，价格回到均线上方 → 回调支撑买入
③ 价格在均线上方回落，未跌破均线即再次上升 → 强势回调买入
④ 价格远离均线向下暴跌，出现负乖离过大 → 超跌反弹买入

卖出信号（四种）：
⑤ 均线从上升转为走平或下降，价格从上方跌破均线 → 趋势反转卖出
⑥ 价格虽突破均线但均线仍下降，价格回到均线下方 → 反弹受阻卖出
⑦ 价格在均线下方反弹，未突破均线即再次下跌 → 弱势反弹卖出
⑧ 价格远离均线向上暴涨，出现正乖离过大 → 获利了结卖出
```

#### 9.2.4 均线的局限性

均线是滞后指标——它反映的是过去的平均价格，对未来的预测能力有限。在震荡行情中，均线会频繁发出错误信号（假金叉、假死叉），导致反复止损。解决方案是结合成交量、MACD等指标交叉验证，或者使用EMA减少滞后性。

### 9.3 成交量分析

成交量是验证价格走势真实性的核心指标。价格可以被少量资金操控，但成交量造假的成本极高。因此，"量在价先"是技术分析的铁律——成交量的变化往往领先于价格变化。

#### 9.3.1 量价关系的完整解读

| 量价关系 | 含义 | 操作建议 | 底层逻辑 |
|---------|------|---------|---------|
| 量增价升 | 多方主导，买盘积极入场 | 持有或加仓，趋势健康 | 资金持续流入推动价格上涨 |
| 量增价跌 | 空方主导，恐慌性抛售 | 减仓或止损，不抄底 | 资金加速出逃，价格加速下跌 |
| 量缩价升 | 上涨缺乏资金支撑 | 警惕见顶，控制仓位 | 仅存量资金推升，后续乏力 |
| 量缩价跌 | 抛压减弱，下跌动能衰竭 | 观望为主，等底部确认 | 卖方不愿割肉，但买方也不愿入场 |
| 地量地价 | 成交极度萎缩 | 阶段性底部可能形成 | 该卖的已卖完，供给枯竭 |
| 天量天价 | 成交极度放大 | 阶段性顶部可能形成 | 最后的疯狂，供需即将逆转 |

#### 9.3.2 关键成交量形态

**放量突破**：股价突破重要阻力位（如前期高点、密集成交区、下降趋势线）时，成交量应放大至前期均量的2倍以上。量越大说明突破的共识越强，越不容易回落。如果缩量突破，大概率是假突破。

**缩量回调**：上涨过程中股价回调但成交量逐步萎缩，说明持仓者不愿卖出（看好后市），抛压很小。这种回调是健康的洗盘行为，通常在均线支撑位附近结束，是加仓的良机。

**放量滞涨**：成交量大幅增加但股价不再上涨甚至微跌，说明上方有大量卖盘，买方力量不足以消化。这是最危险的顶部信号之一，尤其在连续上涨后出现。

**底部堆量**：股价在低位长期横盘，成交量逐步温和放大，说明有资金在暗中吸筹。一旦突破盘整区间上沿，往往开启一段中级上涨行情。

#### 9.3.3 量价分析的进阶指标

# 1. 量比 = 当日成交量 / 过去5日平均成交量
# 量比>2 为放量，量比<0.5 为缩量
volume_ratio = current_volume / avg_volume_5d

        elif close[i] < close[i-1]:

            obv_values.append(obv_values[-1])

### 9.4 趋势判断与趋势线

#### 9.4.1 趋势的定义与分类

道氏理论将趋势分为三个级别：

| 趋势级别 | 持续时间 | 别名 | 判断依据 |
|---------|---------|------|---------|
| 主要趋势 | 1年以上 | 大趋势 | 周K线或月K线的高低点 |
| 次要趋势 | 3周至数月 | 中期调整 | 日K线的回调或反弹 |
| 短暂趋势 | 数天至3周 | 日常波动 | 分钟K线的即时波动 |

**趋势的结构特征**：

```text
上升趋势结构：
                    高点3
                   / 
        高点2    /
       /        /
高点1 /    低点3
  \  /    /
   低点2 /
     \  /
   低点1
特征：高点逐步抬高，低点逐步抬高

下降趋势结构：
高点1
  \  
   \    高点2
    \  / \
     低点1 \    高点3
        \  \  /
         \  低点2
          \  \
           \  低点3
特征：高点逐步降低，低点逐步降低
```

#### 9.4.2 趋势线的绘制与应用

**绘制规则**：

- **上升趋势线**：连接两个或多个依次抬高的有效低点。至少需要两个低点确认，第三个低点触碰趋势线并反弹则趋势线更可靠
- **下降趋势线**：连接两个或多个依次降低的有效高点。同样的确认逻辑
- **有效突破的判断标准**：收盘价（而非盘中价格）突破趋势线幅度超过3%，且连续2-3日站在趋势线另一侧

**趋势线的斜率意义**：

- 斜率较陡的趋势线容易被突破，持续时间有限
- 斜率平缓的趋势线支撑/压力更强，一旦突破意义更大
- 45度角的趋势线最为理想——上涨速度适中，既有持续性又不过热

#### 9.4.3 趋势通道

当上升趋势线和与之平行的压力线构成通道时，价格会在通道内运行。交易策略是：在通道下轨买入，通道上轨卖出。一旦价格突破通道上轨，说明上涨加速（利好）；跌破通道下轨，说明趋势可能反转（利空）。

### 9.5 核心技术指标

#### 9.5.1 MACD（指数平滑异同移动平均线）

MACD是趋势跟踪与动量指标的结合，由Gerald Appel于1970年代发明。其核心是通过两条EMA的差值（DIF）和DIF的移动平均（DEA）来判断趋势方向和动量强弱。

**计算公式**：

```text
DIF = EMA(12) - EMA(26)      快速线
DEA = EMA(DIF, 9)             慢速线（信号线）
MACD柱 = (DIF - DEA) × 2      红绿柱
```

**MACD的四大信号**：

| 信号类型 | 具体描述 | 操作含义 | 可靠度 |
|---------|---------|---------|--------|
| 金叉 | DIF从下方穿越DEA | 买入信号，零轴上方更强 | ★★★ |
| 死叉 | DIF从上方穿越DEA | 卖出信号，零轴下方更弱 | ★★★ |
| 零轴上方金叉 | DIF和DEA均>0时金叉 | 强势买入信号，上涨趋势加速 | ★★★★ |
| 底背离 | 股价创新低但DIF不创新低 | 下跌动能衰竭，反转信号 | ★★★★★ |

**MACD背离是最重要的信号**：

```text
顶背离示例：
股价走势：  低 ──── 高 ──────── 更高
MACD走势：  低 ──── 高 ──────── 较低
含义：股价新高但动能减弱，上涨即将结束

底背离示例：
股价走势：  高 ──── 低 ──────── 更低
MACD走势：  高 ──── 低 ──────── 较高
含义：股价新低但下跌动能减弱，底部可能形成
```

```python
# Python计算MACD和背离检测
import numpy as np
import pandas as pd

def calc_macd(close, fast=12, slow=26, signal=9):
    ema_fast = close.ewm(span=fast, adjust=False).mean()
    ema_slow = close.ewm(span=slow, adjust=False).mean()
    dif = ema_fast - ema_slow
    dea = dif.ewm(span=signal, adjust=False).mean()
    macd_bar = (dif - dea) * 2
    return dif, dea, macd_bar

def detect_divergence(close, indicator, lookback=60):
    """检测背离：比较价格和指标的最近两个极值"""
    # 底背离：价格低点更低，指标低点更高
    # 顶背离：价格高点更高，指标高点更低
    prices = close.tail(lookback).values
    ind = indicator.tail(lookback).values
    
    # 简化逻辑：找最近两个低点
    from scipy.signal import argrelextrema
    min_idx = argrelextrema(prices, np.less, order=5)[0]
    if len(min_idx) >= 2:
        i, j = min_idx[-2], min_idx[-1]
        price_lower = prices[j] < prices[i]
        ind_higher = ind[j] > ind[i]
        if price_lower and ind_higher:
            return "底背离"
    return None
```

#### 9.5.2 RSI（相对强弱指标）

RSI由J. Welles Wilder于1978年提出，衡量一段时间内涨幅占总波动的比例，取值范围0-100。

**计算公式**：

```text
RSI(N) = 100 - 100/(1+RS)
RS = N日平均涨幅 / N日平均跌幅
常用参数：RSI(6)短期、RSI(12)中期、RSI(24)长期
```

**RSI的使用方法**：

| RSI区间 | 市场状态 | 操作建议 |
|--------|---------|---------|
| RSI>80 | 超买区 | 警惕回调，考虑减仓 |
| RSI>90 | 极度超买 | 高概率见顶，果断卖出 |
| 50<RSI<80 | 多头区间 | 趋势向上，持有为主 |
| 20<RSI<50 | 空头区间 | 趋势向下，空仓或轻仓 |
| RSI<20 | 超卖区 | 可能反弹，关注买入机会 |
| RSI<10 | 极度超卖 | 高概率见底，分批建仓 |

**RSI背离**：与MACD背离原理相同，RSI背离同样是高可靠度的反转信号。在RSI超买区出现顶背离或RSI超卖区出现底背离，信号最强。

#### 9.5.3 KDJ（随机指标）

KDJ由George Lane发明，通过比较收盘价与一定周期内价格区间的位置来判断超买超卖。

**计算公式**：

```text
RSV = (收盘价 - N日最低价) / (N日最高价 - N日最低价) × 100
K = 2/3 × 前一日K + 1/3 × RSV
D = 2/3 × 前一日D + 1/3 × K
J = 3K - 2D
默认参数：N=9
```

**KDJ信号**：

- **金叉（K上穿D）**：J<20时金叉为强烈买入信号；J>80时金叉可靠性低
- **死叉（K下穿D）**：J>80时死叉为强烈卖出信号；J<20时死叉可靠性低
- **J值极端**：J>100为超买，J<0为超卖
- **KDJ钝化**：在强趋势中KDJ可能长期处于超买或超卖区，此时不能简单按超买超卖操作

#### 9.5.4 布林线（BOLL）

布林线由John Bollinger创建，由中轨（MA20）和上下轨（中轨±2倍标准差）构成价格通道。布林线的独特之处在于通道宽度会随波动率变化自动调整。

**布林线的交易信号**：

| 信号 | 描述 | 含义 |
|------|------|------|
| 价格触及上轨 | 股价到达上轨附近 | 短期超买，可能回调 |
| 价格触及下轨 | 股价到达下轨附近 | 短期超卖，可能反弹 |
| 布林收口 | 上下轨距离缩小 | 波动率降低，即将变盘 |
| 布林开口 | 上下轨距离扩大 | 波动率增加，趋势启动 |
| 中轨支撑 | 回调到中轨获支撑 | 上升趋势健康，可加仓 |
| 中轨压力 | 反弹到中轨受阻 | 下降趋势延续，应减仓 |

**布林线的特殊用法**：

在强势上涨中，价格会沿着上轨运行（"走上轨"），此时不应因为"触及上轨"而卖出。反之，在弱势下跌中价格会沿着下轨运行。判断标准是看布林通道的方向——通道向上倾斜时沿上轨运行是正常的。

```python
# 布林线计算
def bollinger_bands(close, period=20, num_std=2):
    sma = close.rolling(period).mean()
    std = close.rolling(period).std()
    upper = sma + num_std * std
    lower = sma - num_std * std
    return upper, sma, lower

# %B指标：价格在布林通道中的相对位置
# %B > 1 = 价格在上轨之上；%B < 0 = 价格在下轨之下
def percent_b(close, upper, lower):
    return (close - lower) / (upper - lower)

# 带宽指标：衡量布林通道宽度
# 带宽缩小后放大，通常意味着变盘
def bandwidth(upper, lower, middle):
    return (upper - lower) / middle
```

### 9.6 形态分析

价格形态是K线组合在更大尺度上形成的几何图案，是市场心理的可视化呈现。形态分析是技术分析中仅次于趋势分析的第二大分析方法。

#### 9.6.1 反转形态

**头肩顶/底**：

头肩顶是最经典的顶部反转形态，由左肩、头部、右肩三部分组成。左肩和头部形成时伴随放量，右肩反弹时量能萎缩是关键确认信号。颈线（连接两个低点）被跌破后，理论最小跌幅等于头部到颈线的距离。

```text
头肩顶形态：
     头部
    /    \
   /      \      右肩
左肩       \    /    \
/    \      \  /      \    ← 颈线
      \      \/        \
       \    /           ──── 跌破颈线
        \  /
         \/

确认要素：
1. 右肩高度 < 左肩高度
2. 右肩成交量 < 左肩成交量
3. 收盘价跌破颈线
4. 跌破后通常有回抽确认
```

**双重顶（M顶）与双重底（W底）**：

双重顶是股价两次冲高到相近价位后回落的形态，形似字母"M"。第二个顶部的成交量通常小于第一个，且股价跌破两峰之间的谷底（颈线）即确认形态成立。理论最小跌幅等于顶部到颈线的距离。

双重底则是"M"倒过来，形似"W"。第二个底部通常缩量，突破颈线时放量，是强烈的买入信号。

**圆弧顶与圆弧底**：

圆弧形态是缓慢的趋势转换过程，不像头肩形或双顶那样尖锐。圆弧底尤其值得关注——长期下跌后成交量从萎缩到逐步放大，形成"碗状"底部，一旦突破碗口，涨幅往往可观。

#### 9.6.2 持续形态

| 形态 | 特征 | 方向 | 持续时间 | 交易策略 |
|------|------|------|---------|---------|
| 上升三角形 | 高点持平、低点抬高 | 通常向上突破 | 1-3个月 | 突破上边线买入 |
| 下降三角形 | 低点持平、高点降低 | 通常向下突破 | 1-3个月 | 跌破下边线卖出 |
| 对称三角形 | 高点降低、低点抬高 | 方向不定 | 1-3个月 | 等待突破方向确认 |
| 上升旗形 | 快速上涨后小幅下降通道 | 继续上涨 | 2-4周 | 突破旗形上边线买入 |
| 下降旗形 | 快速下跌后小幅上升通道 | 继续下跌 | 2-4周 | 跌破旗形下边线卖出 |
| 矩形（箱体） | 价格在水平区间内震荡 | 方向不定 | 数周至数月 | 突破箱体上/下沿操作 |

### 9.7 支撑与阻力

支撑位是价格下跌时买方力量集中的区域，阻止价格继续下跌；阻力位是价格上涨时卖方力量集中的区域，阻止价格继续上涨。支撑和阻力是技术分析最基础的概念之一。

#### 9.7.1 支撑阻力的识别方法

**识别支撑阻力的五个维度**：

1. **前期高低点**：历史上的高点在被突破后变成支撑，历史上的低点在被跌破后变成阻力
2. **整数关口**：如沪指3000点、股价100元等心理关口，散户的挂单行为会在这里集中
3. **密集成交区**：成交量在某个价格区间高度集中，意味着大量持仓者的成本在此区域
4. **均线位置**：MA20、MA60等常用均线经常成为动态支撑/阻力
5. **缺口位置**：跳空缺口的上沿或下沿是天然的支撑/阻力

#### 9.7.2 支撑阻力的角色互换

这是技术分析中最实用的规律之一：**被有效突破的支撑位变成阻力位，被有效突破的阻力位变成支撑位**。

```text
角色互换示意：

阻力位 ─────── A ────────── 价格在A处多次受阻
                 |
                 |  突破（放量确认）
                 ↓
新支撑位 ─────── A' ───────── 价格回踩A'获支撑

原理：在A价位卖出的人后悔了（卖早了），等价格再回到A'时他们会买入（弥补遗憾），从而形成支撑。
```

#### 9.7.3 支撑阻力的强弱判断

| 强度指标 | 强支撑/阻力 | 弱支撑/阻力 |
|---------|------------|------------|
| 接触次数 | 被测试3次以上 | 仅测试1-2次 |
| 成交量 | 接触时成交量大 | 接触时成交量小 |
| 时间跨度 | 形成于数月前 | 形成于数周前 |
| K线反应 | 接触后迅速反弹/回落 | 缓慢反应 |
| 距离当前价 | 距离近 | 距离远 |

### 9.8 多周期分析

不同时间周期的图表展现的是不同层级的市场参与者行为。综合多个周期分析可以避免"只见树木不见森林"的错误。

#### 9.8.1 多周期分析框架

```text
分析顺序（先看大再看小）：
月线 ──→ 周线 ──→ 日线 ──→ 60分钟 ──→ 15分钟
大方向     中期趋势    短期趋势    波段节奏    精确入场

核心原则：大周期定方向，小周期找入场
- 月线看牛熊（决定是否参与）
- 周线看趋势（决定做多还是做空）
- 日线看波段（决定何时进场）
- 分钟线看时机（决定精确买卖点）
```

#### 9.8.2 周期冲突的处理

当不同周期信号矛盾时，遵循以下优先级：

- **大周期优先**：周线看涨但日线看跌，以周线为主，日线回调是买入机会
- **同方向共振**：周线、日线、60分钟线同时发出买入信号，是最强的买入时机
- **小周期服务于大周期**：大周期趋势向上时，只在小周期的买入信号处操作

### 9.9 多指标综合分析

单一指标都有其局限性，实战中需要将不同类型指标组合使用以互相验证。

#### 9.9.1 指标组合策略

**趋势指标 + 动量指标 + 成交量**的三角组合是最实用的框架：

| 组合方案 | 趋势指标 | 动量指标 | 辅助指标 | 适用场景 |
|---------|---------|---------|---------|---------|
| 经典组合 | MA20+MA60 | MACD | 成交量 | 中线波段交易 |
| 灵敏组合 | EMA12+EMA26 | KDJ | RSI | 短线交易 |
| 稳健组合 | 布林中轨 | RSI(14) | OBV | 趋势跟踪 |
| 专业组合 | 多均线系统 | MACD+KDJ | 布林+成交量 | 全方位分析 |

#### 9.9.2 信号共振法

当多个指标同时发出相同方向的信号时，该信号的可靠性显著提高：

```python
# 多指标信号共振示例
def signal_strength(signals):
    """
    信号强度计算
    signals: 字典，key为指标名，value为1(看多)/-1(看空)/0(中性)
    """
    bullish_count = sum(1 for v in signals.values() if v == 1)
    bearish_count = sum(1 for v in signals.values() if v == -1)
    total = len(signals)
    
    if bullish_count >= total * 0.7:
        return "强烈看多（多数指标共振）"
    elif bearish_count >= total * 0.7:
        return "强烈看空（多数指标共振）"
    elif bullish_count > bearish_count:
        return "偏多（方向不统一，轻仓）"
    elif bearish_count > bullish_count:
        return "偏空（方向不统一，减仓）"
    else:
        return "中性（观望）"

# 示例：6个指标的信号
signals = {
    "MA": 1,      # 均线多头排列
    "MACD": 1,    # MACD金叉
    "RSI": 1,     # RSI从超卖回升
    "KDJ": 0,     # KDJ中性
    "BOLL": 1,    # 触及布林下轨反弹
    "VOL": 1,     # 放量
}
# result = "强烈看多（多数指标共振）"
```

### 9.10 常见技术分析误区

技术分析用得好可以提高交易胜率，但以下误区会让分析结果适得其反：

**误区一：过度优化参数**

很多交易者花大量时间寻找"最优"参数组合（如MA的最佳周期、MACD的最佳参数），结果在历史数据上表现完美但实盘一塌糊涂。原因是过度拟合——参数是针对历史噪声优化的，对未来无效。解决办法是使用通用参数（如MACD 12/26/9、RSI 14、MA 20/60），不追求参数最优。

**误区二：只看一个指标**

仅凭MACD金叉就买入、仅凭RSI超买就卖出，这种单一指标决策的胜率通常不超过50%。任何指标都有其有效和失效的市场环境，多指标交叉验证是基本纪律。

**误区三：忽略市场环境**

牛市中技术分析偏多操作成功率高，熊市中偏空操作成功率高。在错误的市场环境中使用正确的技术信号也会亏损。操作前先判断大环境，再用技术分析找时机。

**误区四：形态画得太主观**

为了"证明"自己的观点而强行在K线图上画出头肩顶或双底，是确认偏误的典型表现。客观的做法是：如果需要费力才能看出某个形态，那这个形态大概率不存在。

**误区五：把技术分析当预测工具**

技术分析的本质是概率分析，不是确定性预测。任何技术信号都是"在一定概率下价格可能往某个方向走"，而不是"价格一定会涨/跌"。因此，止损和仓位管理永远是交易的底线，不能因为技术分析"看好"就满仓操作。

**误区六：忽略基本面**

纯技术分析在短线交易中可能有效，但中长线投资必须结合基本面。一只基本面恶化的股票，技术形态再好看也可能是"死亡反弹"。技术分析选股的最佳实践是：先用基本面筛选优质标的，再用技术分析选择买卖时机。

### 9.11 实战案例分析

#### 案例一：MACD底背离 + 量价配合抄底

```text
场景：某消费股经历4个月下跌，从35元跌到18元

技术信号：
① 股价第二次下探18元，但MACD的DIF值高于前次低点 → 底背离
② 第二次下探时成交量仅为前次低点时的60% → 缩量下跌
③ 在18元附近出现锤子线+看涨吞没组合 → K线底部形态
④ RSI在28附近，处于超卖区 → 超卖确认

操作方案：
- 信号共振确认后，在18.5元建仓30%
- 股价站上MA5后加仓20%
- 股价突破下降趋势线（约20元）再加仓20%
- 股价站上MA60后持有，设MA20为移动止盈线

结果：股价在3个月内反弹至28元，收益约50%
```

#### 案例二：布林收口 + 均线粘合选股

```text
场景：寻找即将变盘的股票

选股条件（用Python实现）：
1. 布林带宽处于近60日最低水平（波动率极度压缩）
2. MA5/MA10/MA20/MA60四条均线距离<2%（粘合状态）
3. 最近5日成交量持续温和放大
4. 股价处于布林中轨上方（偏多方向）

原理：波动率压缩→均线粘合→变盘在即→
     成交量放大暗示方向→中轨上方暗示偏多
```

```python
import pandas as pd
import numpy as np

def screen_consolidation(df, period=60):
    """筛选布林收口+均线粘合的股票"""
    close = df['close']
    
    # 布林带宽
    sma20 = close.rolling(20).mean()
    std20 = close.rolling(20).std()
    bandwidth = (2 * 2 * std20) / sma20  # 带宽
    
    # 均线
    ma5 = close.rolling(5).mean()
    ma10 = close.rolling(10).mean()
    ma60 = close.rolling(60).mean()
    
    # 均线粘合度（最大均线与最小均线的差距）
    ma_max = pd.concat([ma5, ma10, sma20, ma60], axis=1).max(axis=1)
    ma_min = pd.concat([ma5, ma10, sma20, ma60], axis=1).min(axis=1)
    ma_spread = (ma_max - ma_min) / close
    
    # 条件判断
    latest_bw = bandwidth.iloc[-1]
    min_bw = bandwidth.tail(period).min()
    is_narrow = latest_bw <= min_bw * 1.05  # 带宽接近最小值
    is_glued = ma_spread.iloc[-1] < 0.02    # 均线差距<2%
    above_mid = close.iloc[-1] > sma20.iloc[-1]  # 在中轨上方
    
    return is_narrow and is_glued and above_mid
```

### 9.13 小结

技术分析是一门经验科学，它的价值不在于预测未来，而在于**提高决策的概率优势**。掌握技术分析需要经历三个阶段：

1. **入门阶段**：学会看K线、均线、成交量的基本含义，理解趋势和形态
2. **进阶阶段**：熟练使用MACD、RSI、KDJ等指标，掌握背离和多指标共振分析
3. **精通阶段**：形成自己的分析框架，能在不同市场环境中灵活运用，并结合基本面和资金面做综合判断

最后，技术分析最大的敌人不是市场，而是交易者自己的情绪和偏见。再好的技术分析，如果没有纪律性执行（严格止损、控制仓位、不追涨杀跌），也无法转化为实际收益。记住：**技术分析是工具，交易纪律才是武器**。
